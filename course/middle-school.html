Beauty and Joy of Computing for Middle School

Welcome to BJC! This class teaches computer programming using the Snap! programming language, which lets you do powerful programing without worrying about punctuation as you would in a text-based language. You'll build projects about story-writing, music, secret messages, artificial intelligence (AI), and more! And you'll learn to use electronics to connect your code to the real world.
