Make an Interactive Face

So far, the face changes on its own, but you can control when it changes—for example, by using the micro:bit's light sensor. The light sensor can detect different levels of light, and you can program the micro:bit to change the face depending on how much light the sensor detects.

  1. Use an if block together with a light level block to change the micro:bit's facial expression based on the light it detects.
