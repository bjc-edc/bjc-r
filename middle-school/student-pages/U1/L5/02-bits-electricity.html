Storing Numbers in Bits

: Bit

The word "bit" is an abbreviation for binary digit.

We can also call each place a "bit" (short for binary digit), so 1010 is a 4-bit number. A number that's only a few digits long in decimal may be many bits long in binary, but that's okay, because bits are the only thing that computers know how to store.

Okay, so your computer is constantly converting to binary behind the scenes (sneaky computer!). Let's see what that actually looks like ourselves though.

Storing Bits as Electricity

At this point, you might be wondering if you could dissect your computer and discover a bunch of 0s and 1s in the carcass. That'd be cool/creepy, but actually, computers don't really store the digits 0 and 1.

Computers are electronic devices, so they store binary data using electricity. Generally, a 1 is stored as a positively charged electric signal and a 0 is stored as no charge at all.
