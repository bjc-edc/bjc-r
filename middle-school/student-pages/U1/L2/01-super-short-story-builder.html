Super-Short Story Builder

In this activity, you'll use functions to write one-sentence stories.

Play with a Super-Short Story Builder

  1. Open this Super Short Stories project.

Every Snap! program is made up of blocks, and those blocks tell the computer what to do. There are lots of blocks in Snap!, and you can even make your own blocks. For this first project, we're hiding most of the blocks, but there is one already in the Scripts area (the central space of the Snap! window) for you to play with.

  1. Click the Super Short Story block, and read the bubble that appears.

There are multiple types of blocks in Snap. This Super Short Story block is a reporter block. That means that it does some sort of computation and then "reports" the result to the rest of the program. In Snap! reporter blocks are oval-shaped.

When you click a block, the computer runs its code. You'll see the code behind this block in an upcoming activity. For now, notice that when you click the block, you see the reported result in a bubble, like this:

Many blocks accept input parameters that customize what they compute, and those parameters show up as blank boxes. This block reports a story based on the input parameters (a feeling, a job, a verb, etc.). You can change the inputs to change the output (in this case, the story).

  1. Change some of the input parameters to the Super Short Story block, and click it again.
  2. What happens to the story in the bubble?

Snapping Blocks Together

Snap! is called Snap! because you can snap blocks together. Here a random feeling block that has been snapped into the Super Short Story block.

  1. Find one of these random blocks in the palette on the left side of the screen, and click it several times.
  2. What was reported?

These blocks don't take any input parameters, but since they are reporters, they still report an output. You can use the output of one block as the input to another block.

  1. Drag some of the random blocks into the Super Short Story block, and click the block a few times.
  2. What happens to the story reported by the block?
  3. Make sure you are logged in to Snap! by checking that your name appears in the Snap! Cloud menu ().
  4. Save this project to your Snap! account by choosing "Save" from the Snap! File menu (it looks like a piece of paper with a corner folded: ).

Be sure to save your projects after an important change to your project and, if you are using a shared computer, whenever you're done for the day.

In this activity, you learned how to run a block by clicking it and how to change a block's input parameters by typing different text into the input slots or by dragging other blocks into the input slots. You've seen that, either way, changing the inputs that are given to a function will change the output that it reports.
