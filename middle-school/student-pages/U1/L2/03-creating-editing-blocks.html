Creating and Editing Blocks

In this activity, you'll edit the structure of the story itself and create your own random blocks.

Editing a Block

Some blocks (like join) are built into Snap! and you can't change them, but Snap! programmers (like you!) can also make and edit their own custom blocks. The random item from a list blocks in this project are custom blocks that you can edit.

  1. Choose one of the random blocks, right-click it, and select "edit…" from the menu that appears.
    An animation would be nice here when we have time. --MF, 8/11/21

This custom block reports a random item from a list of jobs.

  1. Change some of the items in the list for the block you are editing. (That is, change some of the colors to other colors, some of the jobs to other jobs, or some of the places to other places, etc.)

You can add additional items to the list or reduce the number of items in the list by clicking the arrows at the end of the list block (◄ ►).

  1. When you are finished, click the "OK" button, and try clicking the block that you edited several times to make sure that it reports a random item from the revised list. (If it doesn't work right, edit the block again and also edit another one of the random blocks at the same time. Compare the code for your edited random block that isn't working with the random block that is working.)
  2. Once you have a working revision to a random block, try it out in your Super Short Story block.
  3. Edit a few other random blocks to customize your project further.

Editing the Story Structure

The join block joins two strings of text into one (without a space), so you can use it to create a plural word. There is also a join words block that joins two strings of text into one with a space between them.

  1. Right-click the Super Short Story block, and select "edit…" from the menu that appears.

The hat block at the top shows you the structure of the block that will be visible from the outside, including the name of any input parameters (feeling, job, action 1, place, action 2, etc.). The code inside the report block below controls what will be reported when the Super Short Story block is clicked. The input parameters can be dragged into the input slots to the code for the block, and the values that they each report to the join words block will vary based on what is typed into the input slots. These input parameters are called local variables because they vary, but they only can be used in this local context.

  1. Change the structure of the inputs to the join words block to change the structure of the story. When you are finished click "Apply" to save your changes, and click the Super Short Story block to test your work.

Creating a Block

Students will also explore creating their own random blocks. You can create one by right-clicking the background of the scripting area, naming a new reporter block, right-clicking to copy the contents of another random block, dragging it in, and editing the contents.
