Keeping Specific Items from Your List

You can already learn some things about the data you have collected by using the keep and length blocks. For example, you might want to know how many students that you surveyed study more than 5 hours a week.

(You can use responses to question to select the data for one question, then use keep to select responses meeting a certain criteria.)

The block takes a list as input and reports the length of that list.

If the input to length of list is a list of lists, it will report the number of lists in the list. If you want to know the length of one of your lists inside the list, you first need to select the data you want to ask the length of by using responses to question.

As you know, predicates can be used with conditionals to decide when to do something; they can also be used with keep to decide which things to keep. The block takes a list and a predicate as input, and it reports a new list keeping only the items from the input list that make the condition described by the predicate true.

For example, the following expression will find words from the words list whose first letter is v. The blank input slot is where each item of the list goes to decide if it will be kept.
