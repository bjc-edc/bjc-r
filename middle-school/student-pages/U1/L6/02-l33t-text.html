Make a New Stylized Text Block

To show you how a text transforming block actually works, we'll make a block to transform text messages into l33t text, a kind of writing that replaces the letter "e" with the number 3.

  1. Click in the palette. In the dialog that pops up, type the block name "l33t" and select "Reporter" as the type. Like all the other text styling blocks we used, this one will take in a value and report a value back. You should see a block editor like this one:
  2. This block needs to take a single input, the text message to transform. Click the + sign at the end of the block header and select "Input name" in the dialog that pops up. Type "text" for the input name.
  3. The first step in transforming the text input is to split it into pieces to process. Since this style of writing cares about individual letters, we need to split by letter. Drag in the split block. Replace the default "hello world" with the text input by dragging it from the block header, and select "letter" from the dropdown.

The split block will turn the text into a list of letters, one row for each letter in the message. For example, here's how it would process the message "hey":

  1. The next step is to use a map block, to turn that list of letters into a list of transformed letters. Drag the map block into the editor, and drag the split block onto the list icon.

Now we need to figure out what to put in the first slot. For the example message of "hey", this is what we'd want to happen:

We can describe that mapping with a simple conditional: If the letter is e, then it should be mapped to 3, otherwise the letter stays the same.

  1. Drag an if-then-else block into the empty slot of the map block:
  2. Drag an = block into the if slot. Keep the first slot empty, since that will get automatically filled in with the current letter being mapped, and type "e" in the second slot.
  3. Now type an "e" in the then slot, and leave the else empty. Any empty slot in a map will be assumed to be the current value being transformed. (In this case, the current letter).

We're almost done! We now have a list of transformed letters. This block should report back a string, however. Fortunately, it's easy to turn a list into a string.

  1. Drag the join block into the editor. By default, it has two slots filled in with strings. Remove the second slot and fill the first one with your map block.
  2. Finally, drag that join block into the report slot. Your l33t block is complete!
  3. Of course, we need to make sure it actually works. Drag a few l33t blocks into the scripts tab and try out some messages. You can also try them inside the draw conversation block, to see them in all their iPhone SMS glory.

Make up Your Own Style!

What other blocks could you make like this one? Here are some ideas:

  • Replace a different letter than "e"
  • Replace more letters, by nesting more if's inside the original if.
  • Replacing a letter with an emoji.
  • Replacing a letter with multiple letters.

  1. To easily make a new block that is similar to l33t, you don't need to code all of that from scratch. Instead, just right-click on the block name and select "duplicate block definition". That'll bring up the block editor again:

    You'll want to click "l33t" to change the name, and delete the "(2)" at the end.
  2. Now you can change what's in the if-then-else to make your own styling block.
  3. Once you're done, try out your new block, use it in a conversation, and share your work with your classmates!
