A Brief History of Emojis

Emojis originally came from Japan, where they were built into mobile phones as a way of adding little pictures to text messages. Many of the emojis that we use today originated from that first set of Japanese emojis:

Emojis became very popular, so people from multiple countries came together to figure out how to bring emojis into software all over the world. Emojis became standardized, which means that there is a detailed document that describes how emojis should be transferred digitally, and that every emoji corresponds to a particular string of binary digits (0s and 1s).

For example, üíñ is known as the "sparkling heart" emoji and is stored in binary as 11110000:10011111:10010010:10010110. A computer user doesn't have to know the binary of course, but computer programmers have to agree on it so that emojis are the same across devices. Imagine if üíñ got turned into üíî! That could be very awkward indeed.

For a longer history of emojis, check out this talk by a professional emoji historian. (Yes, that's a thing).

Understanding the Emojify Block

  1. Open your "Texting Time" project, if it isn't open already.
  2. Right-click on the emojify block and select "Edit". You should see this in the block editor:
  3. How does this block work?

    • The emojify block is a reporter block, so it uses a report block to send back a computed value to the main program.
    • The report block has a nested join block. A join block joins together multiple strings of text into one long string of text.
    • The join block has a nested map block. A map block performs the same operation on every item in a list and reports a new list containing those results. The first input is the operation, the second input is the list.
    • The first input to the map block is another join block that joins the result of calling the emojify word block with a space character.

    • The second input to map is the split block that splits the input text into a bunch of words.

So for the phrase, "I love rain", that first gets split into the list "I", "love", "rain", then the emojify word block gets called on each word, resulting in the list "I", "‚ù§Ô∏è", "‚òîÔ∏è", and then finally, the join block turns that list into "I ‚ù§Ô∏è ‚òîÔ∏è".

Add More Emojis

  1. Add more word-emoji pairs by editing the emojify word block:
    1. Add a new item by clicking the right arrow at the end of the list block. You should see a new input slot appear at the end of the list.
    2. Duplicate one of the nested word-emoji lists by right-clicking and selecting "duplicate".
    3. Drag the duplicated block into the newly added blank.
    4. Change the word and emoji by typing in the blanks. You can paste emojis from emojifinder.org or emojipedia.org.
  2. Test out your change! There are two ways to test it:
    1. Drag an emojify word block into the main area, and type the new word in the blank. Click to see the new emoji.
    2. Drag an emojify block into the main area, and type any sentence that contains that word. Click to see the sentence with the new emoji included.
  3. You can continue to add more word-emoji pairs the same way.

Design an Emoji

This also was a Worksheet with three sections: Design, Possible meanings, Example usage. --MF, 9/30/21

The Unicode Consortium is the organization in charge of emojis (as well as all the other characters in Unicode, the standard way of storing text in binary). They actually accept proposals from the public for new emojis. Check out their proposal process.

  1. What emoji do you think the world needs? Maybe an emoji that would help you better express yourself on digital devices, or an emoji for your favorite hobby, or an emoji that represents a culture you come from.
  2. Draw the emoji on a sheet of paper or search the web for images that fit the ideas in your head. Think about how the emoji would look like in color as well as in black & white; all emojis must be available in both forms, since not all digital devices support color. And of course, make sure your emoji isn't in Unicode! See the official emoji list.
  3. Consider these questions from the official emoji proposal process:
    • Can the emoji be used in multiple contexts? Does it have literal as well as metaphoric meanings?
    • Could the emoji be used in a sequence with other emojis to create meanings that aren't already possible?
    • How different is the emoji to existing emoji? What's the most similar emoji?
  4. Present your emojis to your classmates and the whole class. If you feel passionately about some of your proposals, perhaps you'll decide to go through the formal proposal process. Five years from now, you could be typing an emoji that you yourself proposed!
