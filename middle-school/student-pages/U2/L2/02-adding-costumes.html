Adding Costumes and Backgrounds

In this activity, you'll add costumes to your Sprites and a background to the Stage.
You can use the ones built-in to Snap!, import pictures from the Internet, or draw your own.

The Stage is what users of your project see. One or more sprites (characters) live on and can talk and move around on the stage. Sprites begin with an arrowhead costume (), but you can change a sprite's costume (the picture that it wears) by choosing "Costumes..." the Snap! File menu ().

Will you use a costume that is built into Snap!, an image from the Internet? Or will you create your own in Snap!, another application, or on paper?

Adding a Stage Background

  1. Change the background on the stage.

    Adding a Built-in Stage Background

    1. In the lower right corner of the Snap! window, select the stage icon (shown right).
    2. Choose "Backgrounds..." from the File () menu.
    3. Choose a background.
    4. Click "Import."
    5. Click "Cancel" to close the dialog box.

  1. Add more backgrounds if you want.
  2. Change the background to the first background that you want to appear.

    Changing a Stage Background

    1. If it isn't selected already, Select the stage icon in the lower right corner of the Snap! window.
      2. The "Backgrounds" tab isn't always visisble. It replaces the "Costumes" tab when the stage icon is selected.
    2. Click the "Backgrounds" tab above the scripting area.
    3. Select a different background.

Adding a Costume

  1. Change the costumes for your first sprite. (You can add more sprites later.)
    Changing a costume is almost like changing a stage background! The only differences are in bold below.

    Adding a Built-in Sprite Costume

    1. In the lower right corner of the Snap! window, select the sprite icon (shown right).
    2. Choose "Costumes..." from the File () menu.
    3. Choose a costume.
    4. Click "Import."
    5. Click "Cancel" to close the dialog box.

  1. Add more costumes (for this sprite) if you want.
  2. Change the costume for this sprite to the first costume that you want it to have.

    Changing a Sprite Costume

    1. Select the sprite icon if it isn't selected already.
      2. The "Costumes" tab isn't always visisble. It replaces the "Backgrounds" tab when a sprite icon is selected.
    2. Click the "Costumes" tab above the scripting area.
    3. Select a different costume.
  1. Fix any issues with your costumes.

Adding and Naming a Sprite

You can use a photograph or picture from the Internet by dragging the image onto the Snap! window.
  1. If your project needs more than one sprite, add more.

    Adding a Sprite

    • Clicking on the button just below the stage.
      The new sprite, automatically named "Sprite(2)", will appear on the stage. The pictures of the sprites beneath the stage let you select the sprite you want to work with.
  2. If you like, give names to your sprites so you can keep track of them.

    Naming a Sprite

    • Look for the name in a gray box above the scripting area. Click inside the name and you'll be able to edit it.

Try Your Own Image

These are optional extra steps for anyone who has access to a webcam or has a own photo of their own.

  1. If you prefer, add your own background image.

    Drawing an Image in Snap!

      You can read more about the drawing tools in the Snap! Reference Manual. Look in the index for "Paint Editor".
    1. Select the stage icon, and then click the "Backgrounds" tab.
    2. Click the "paint" icon ().
    3. Create your background.
    4. Click "OK."
