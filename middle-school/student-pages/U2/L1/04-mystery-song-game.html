Mystery Song Game

Dan wants a "Guess the song" activity. Maybe even a song player that plays any number of songs and then you guess what it was. --MF, 1/4/22
In this activity, you will create a game where the user has to guess the name of the song played.
In this activity,
  1. Play your song for someone else, and see if they can guess what it is; guess the song for someone else.
Have them use command ask and command if together with say to test the user's guess. We could even do and/nested if here to accommodate alternate spellings/capitalizations.

There are command block versions of two blocks you've already seen:

  • You've used the reporter. Snap! also has a built-in command version:
    Learn more about the difference between the reporter if then else and the command if else.

    In Unit 1 Lab 6 Activity 2: L33t Text, you used to decide what the computer should report. You can use the command version to decide what the computer should do. Like the reporter version, the command if else block takes three inputs:

    1. A testing condition (such as )
    2. What to do if it's true
    3. What to do if it's false

    The command block, however, performs a set of commands depending on the result of the condition rather than reporting an output.
