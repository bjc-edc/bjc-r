In this activity, you will create a command block that plays a song for a user to guess.

You've mostly used reporter blocks so far, blocks that compute some value and report it back. The play ♪ block is a command block, a block that tells the computer to do something, such as to play a sound or to draw, move, or change something.

You learned about reporters in Unit 1 Lab 1 Activity 4: Say Hello to Snap!.

A reporter block does some sort of calculation and then "reports" the result. Reporters have an oval shape.

You learned about predicates in Unit 1 Lab 6 Activity 2: L33t Text.

A predicate is a special kind of reporter that asks a true/false question and reports either true or false. Predicates have a hexagon shape.

You'll use commands much more in Unit 2.

A command block tells the computer to do something without reporting a value. Commands have a puzzle shape and can be snapped together one after another.

Unlike a reporter, a command block does not report anything back, so you can't use the result of a command block as the input to another block—there is no result!
