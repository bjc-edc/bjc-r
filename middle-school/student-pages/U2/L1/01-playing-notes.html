    The pitch tells the computer which note to play. It can be any whole number from 0 to 127. The number 60 represents the note middle C. If you click the triangle, a piano menu will open, and you can hear the note you are selecting.

    The number of beats tells the computer how long to play the note. The values work just like in music: 1 beat is a whole note, 0.5 beats is a half note, 0.25 beats is a quarter note, etc.

  3. Change the inputs to the play ♪ block and then click it, and then change the inputs and click it again to see how the note changes.
    4. Tip: Right click a block and select "help" to see an explanation of how to use it.
  4. Attach several play ♪ blocks together one after another to begin creating a song.
