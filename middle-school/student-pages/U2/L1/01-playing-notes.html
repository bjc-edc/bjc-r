In this lab, you'll teach the computer to play some simple songs and move those songs up and down the musical scale.

Playing singles notes

  1. First download the starter project: Song Player.

Snap provides a built-in block for playing notes, the block.

This block takes two input parameters:

  • The first input parameter is the note to play, a number from 0 to 127. If you're familiar with music, the number 60 represents the Middle C. When you click on the number in the block, a little piano will actually pop up and you can select a note that way.
  • The second input parameter is how long to play the note for. It defaults to 0.5 beats, but you can increase it or decrease it however you'd like.

Tip: You can also right click the block and select "help" to see an explanation of how to use built-in blocks.
  1. Play around with the parameters, clicking the block each time to see how the note changes. What happens if you pick a really small number, like 0? What about a big number like 127?

Command blocks

In the first lab, we mentioned that there are multiple kinds of blocks in Snap. We've mostly been using reporter blocks so far, blocks that compute some value and report it back. The play note block is an example of a command block, a block that commands the computer to do something. In this case, it commands the computer to play a sound. In other cases, command blocks command the computer to draw something on the screen.

In either case, a command block does not report anything back, so we can't use the result of a command block as the input to another block. There is no result!
