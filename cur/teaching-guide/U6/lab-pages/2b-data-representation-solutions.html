  • Page 4: Floating Point.
    • Take It Further:
      This number line from 0 to 1000 in increments of 0.01 may give you an idea.

      The spacing is equal in increments of 0.01. There are 100 values from 0 to 1 and 900×100=90,000 values between 0 and 900. As each value is a terminating decimal, each of these represents a fractional value (i.e. a rational number). There are 99 integer values from 901 to 999 that are not representable. A strength of this system is that you can capture and represent a sizable number of real numbers (90,000 of them) accurate to the hundredths digit. The weakness is that you cannot capture every real number (for example, 789.123) between 0 and 900. The system is finite and have limited precision.
