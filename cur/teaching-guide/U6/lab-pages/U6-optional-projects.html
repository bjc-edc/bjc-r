Optional Projects

Pacing

Expected times to complete the optional project follow, but these times will vary significantly depending on your students:
  • 60-90 minutes (2-3 class periods) on Binary Timer Project

Lab Pages

  • Page 1: Binary Timer Project.
    • Learning Goals:
      • Understand algorithm(s) that are used to convert from decimal notation to binary notation.
      • Develop intuition about how decimal and binary representations relate to each other.
      • Implement timer that displays elapsed time in both decimal and binary notations.
      • Write code to supplement and extend existing code.
    • Tips:
      • This project will require students to produce code that needs to work in tandem with some pre-existing code. If students continue to become professional programmers, this skill to work with and extend existing code will be vital. Make sure they study existing code blocks in this project before generating new code.
