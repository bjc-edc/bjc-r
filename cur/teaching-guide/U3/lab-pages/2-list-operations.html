Programming Lab 2: Three Key List Operations

This lab introduces students to the map, keep, and combine list blocks. Map performs an operation on each item of a list and reports a modified list. Keep filters a list and returns a list of all items from the orginal list that match a given requirement. Combine performs an operation (such as addition) with every item of a list and returns a single result (such as the sum).
