Introduction to Lab 1

  • Analyze a script: Project the scripts and and ask students to describe what they will do. These should be familiar.
  • Analyze a task: Project this animated image

    and ask students to describe the behavior they are seeing.

    You can expect that initial contributions will be fairly general—"The computer is writing the word 'mouse'" or even "It's writing in two colors." It's sensible to start with the global behavior. Continue asking for contributions, that add more precision or detail: "It looks like the writing changes to red when it gets to the top half of the screen" or "the sprite and the mouse are moving together" or....

    Observations like these help the student recognize what features might be needed in a program that can create this behavior.
