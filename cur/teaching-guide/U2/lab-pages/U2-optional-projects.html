Optional Projects

Pacing

Expected times to complete each optional project follow, but these times will vary significantly depending on your students:
  • 30-40 minutes (about 1 class period) on Automated Fortune Teller
  • 40-60 minutes (1-2 class periods) on Egyptian Motif
  • X-X minutes (X-X class periods) on Mastermind
  • 40-60 minutes (1-2 class periods) on Tic Tac Toe Using Sprites

Lab Pages

  • Page 1: Automated Fortune Teller.
    • This less difficult project aims to build more experience in using the list data structure. If exciting costumes and sounds are incorporated, the project can become very creative and engaging.
  • Page 2: Egyptian Motif.
    • This step-by-step project aims to develop intuition about the concept of a variable in the context of Islamic artistic design. The project ties directly to Unit 1 Lab 3 where pinwheel and rectangle blocks were developed. It could be recommended to students with an interest in art and the Islamic culture. A great reference for your students is "Visual Modeling with Logo" by James Clayson (MIT Press).
  • Page 3: Mastermind.
  • Page 4: Tic Tac Toe Using Sprites.
    • Based on the feedback we received from previous years, this year we wanted to give an alternative approach to the Tic Tac Toe project in which all the tedious math is gone and the programmers can focus on the game itself and ultimately explore interesting avenues for the game such as developing simple Artificial Intelligence (AI) to implement a Human versus Computer game in Phase 3 of the project. In this new version of the Tic Tac Toe, the squares of the Tic Tac Toe board are click-conscious sprites, thus eliminating the need for extensive mathematical computations to link mouse click locations to board cell numbers. Even though this approach to Tic Tac Toe will be optional, we hope that you and your students will consider it and take on the challenge of creating something quite profound like Artificial Intelligence (AI) at this first course in Computer Science.
