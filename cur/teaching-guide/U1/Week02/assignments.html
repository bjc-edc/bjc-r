Week 2 Assignments

Optional assignments for class discussion, silent work, or homework (for use anytime during or after the indicated lab):

  • Consider assigning some of the Week 2 discussion prompts as writing prompts.
  • Bug Dance Animation Investigation (Lab 4)
  • You may wish to specifically assign one or more of the optional "Take it Further" problems:
    • draw triangle (Lab 3, Designing Custom Blocks)
    • draw house (using the draw triangle block) (Lab 3, Designing Custom Blocks)
    • Drawing houses of any size (Lab 3, Designing Custom Blocks)
    • Extending draw triangle and draw house to accept a size input (Lab 3, Adding an Input)
    • Creating a draw polygon block (Lab 3, Adding an Input)
    • Nesting polygons with inputs for number of sides and sidelength (Lab 3, Adding an Input)
    • Making Sprites chase each other with the forever block (Lab 4, Commands and Reporters)
    • Playing random notes with repeat and random blocks (Lab 4, Commands and Reporters)
    • Recognizing "less than or equal to" and "greater than or equal to" when built with blocks (Lab 4, Predicates, Conditionals, and Logic)
    • Writing a script to identify leap years (Lab 4, Computer Operations)
  • Encourage students to suggest their own ideas for programming projects based on the material they have learn so far. If a project idea is too advanced for this time, offer suggestions for elements of the larger goal that could be tackled now with a smaller project.
