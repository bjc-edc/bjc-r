Programming Lab 2: Control Commands

Students become acquainted with the snap stage as a coordinate plane.

Prepare:

  • Look through students' posted Kaleidoscope projects and think of the projects you would like to have students see.
  • Prepare a Computing in the News story.
  • Make sure students know the procedure for doing their homework and turning it in, especially if they will do this assignment online.

Students will:

  • Practice pair programming
  • Incorporate x- and y-positions in their scripts
  • Explore and use forever

Openers:

  • Aww, Snap!: Moving forever
    Students reason through code without the use of a computer. The code includes bugs for students to resolve.
  • Make a Prediction: Introduce repeat
    Students reason through the code without the use of a computer. The code may include elements they have not yet used or complicated logic to sort through.
