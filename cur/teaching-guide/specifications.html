Snap! Minimum Specifications

The programming platform used for the Beauty and Joy of Computing curriculum, Snap!, relies on JavaScript that has been standard for over a year. If your system is not up to date, this an cause any number of problems with Snap! (errors when you haven't done anything wrong, sound features not working, etc.)

Please check that your browser is at least as up to date as the versions shown below:

BrowserMinimum Version Required
Chrome43
Safari8.4
Firefox38
Edge12
Opera32
IEDoes not comply with web standards. Do not use.

Please note that sometimes a browser (such as Chrome) may report that it is up to date even though it is far behind the versions shown above. This can happen then the computer's operating system is so far outdated that the browser can no longer update to the current version. In this case, please work with your technical support team to upgrade your system.
