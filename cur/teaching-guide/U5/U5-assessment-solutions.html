  • Problem 1:
    • Algorithm A:

      The outer for loop runs number times, and every time that outer for loop runs, the inner for loop runs number times. So, Algorithm A will report number^2 loops indicating quadratic polynomial (reasonable) time.

    • Algorithm B:

      The for loop runs 10^{number} times. So, Algorithm B will report 10^{number} loops indicating exponential (unreasonable) time.
