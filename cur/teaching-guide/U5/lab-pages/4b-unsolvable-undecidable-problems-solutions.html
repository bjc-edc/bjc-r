  4. Betsy's statement is a paradox; it is a logical contradiction that is neither true nor false. If the statement "The statement I'm making right now is false" is true, then is has to be false because the true statement says it is false. However, if the statement is false, then because it says it is false, it must be true. Neither makes sense.
