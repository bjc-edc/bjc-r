This exercise will give you some more practice working with recursion. This time, we are going to use recursion to determine whether a word is a palindrome.
A palindrome is a word or number that is identical when read both forwards and backwards. Most definitions of a palindrome ignore all punctuation, so we will be doing that in our exercise today as well. Some simple palindromes include "kayak," "radar," and "dad," although sentences, poems, and stories can be palindromic as well. For a more complete list, check out http://www.palindromelist.net/.

Exercise

Create a reporter block that has a single argument (a phrase) and returns either true (if the phrase is a palindrome) or false (if the phrase is not a palindrome).
