En este proyecto, crearás un estudio para diseñar y animar un personaje.

En este proyecto:

  • El usuario utilizará el clic y arrastre del ratón para dibujar un personaje.
  • Las ubicaciones del ratón (x y y) visitadas durante el dibujo se almacenarán en una lista.
  • El usuario moverá el personaje dibujado usando las teclas de flecha arriba, abajo, derecha e izquierda.

  1. Habla con tu compañero sobre cómo diseñarían un proyecto de este tipo. Asegúrate de explorar qué datos deben rastrearse y cómo se pueden emplear listas para realizar un seguimiento de estos datos.
    2. Consejos para guiar tu pensamiento:
    • ¿Cómo puede almacenar las posiciones x e y del ratón a medida que se crea el dibujo??
    • ¿Por qué es importante recordar el orden de estos lugares?
    • Utiliza las teclas de flecha para mover toda la imagen.
    • ¿Cómo se debe actualizar la información de ubicación almacenada en las listas para realizar estos movimientos?
    • ¿Cómo se deben crear los dibujos revisados en función de las ubicaciones actualizadas?
  2. ¡Empieza a codificar! Muchas cosas se aclararán cuando empieces a trabajarlo.

Es posible que desee utilizar el bloque unificar para que tus animaciones se representen más rápidamente.

Si desmarcas la lista de puntos en la paleta Variables, tu animación se ejecutará más rápido porque tu computadora no tendrá que mostrar cientos de coordenadas en el escenario.

Haz una lluvia de ideas con tu compañero sobre algunas posibles extensiones del proyecto. Aquí hay algunas ideas:

  1. Usa la abstracción y revisa tu código para que todas las entradas de la tecla de flecha (arriba / abajo / derecha / izquierda) puedan ser abordadas por un solo bloque con entradas.
  2. Añade imágenes de fondo, sonido/música a la animación de tu personaje.
  3. Proporciona al usuario opciones (como color, grosor del lápiz, etc.) en la fase de dibujo.
  4. Crea una serie de animaciones en tu programa Animation Studio. Captura, combina y edita tus animaciones con un software como Screencast-O-Matic y luego sube tu trabajo a YouTube y compártelo con tus compañeros de clase.
