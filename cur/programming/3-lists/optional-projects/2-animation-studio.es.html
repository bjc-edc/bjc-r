En este proyecto, crearás un estudio para diseñar y animar un personaje.

En este proyecto:

  • El usuario utilizará el clic y arrastre del ratón para dibujar un personaje.
  • Las ubicaciones del ratón (x y y) visitado durante el dibujo se almacenará en una lista.
  • El usuario moverá el carácter dibujado usando las teclas de flecha arriba, abajo, derecha, izquierda.

  1. Discuta con su socio cómo diseñaría un proyecto de este tipo. Asegúrese de explorar qué datos deben rastrearse y cómo se pueden emplear listas para realizar un seguimiento de estos datos.
    2. Consejos para guiar tu pensamiento:
    • ¿Cómo puede almacenar las posiciones x e y del ratón a medida que se crea el dibujo??
    • ¿Por qué es importante recordar el orden de estos lugares?
    • Utilice las teclas de flecha para mover toda la imagen.
    • ¿Cómo se debe actualizar la información de ubicación almacenada en las listas para realizar estos movimientos?
    • ¿Cómo se deben crear los dibujos revisados en función de las ubicaciones actualizadas?
  2. ¡Empieza a codificar! Muchas cosas se aclararán cuando realmente te pongas a trabajar.

Es posible que desee utilizar el warp bloque para que sus animaciones se representen más rápidamente.

Si ustedDesmarque el lista de puntos en la paleta Variables, su animación se ejecutará más rápido porque su computadora no tendrá que mostrar cientos de coordenadas en el escenario.

Haga una lluvia de ideas con su socio sobre algunas posibles extensiones del proyecto. Aquí hay algunas ideas:

  1. Use la abstracción y revise su código para que todas las entradas de la tecla de flecha (arriba / abajo / derecha / izquierda) puedan ser abordadas por un solo bloque con entradas.
  2. Añade imágenes de fondo, sonido/música a la animación de tu personaje.
  3. Proporcione al usuario opciones (como color, grosor del lápiz, etc.) en la fase de dibujo.
  4. Crea una serie de animaciones en tu programa Animation Studio. Capturarlos, combinarlos y editarlos con un software como Screencast-O-Matic y luego sube tu trabajo a YouTube y compártelo con tus compañeros de clase.
