The triangular number function adds all of the whole numbers from 1 through the input number. So triangular number (n) reports 1 + 2 + 3 + \cdots + (n - 1) + n.

The factorial function multiplies all of the whole numbers from 1 through the input number. In other words, (n) ! reports 1 \times 2 \times 3 \times ... \times (n - 1) \times n.

The code using recursion is shorter and simpler to look at; programmers would call it "more elegant." But there is an even easier way. You can just combine the elements of a list using whichever operation you want to use (addition or multiplication).

The block takes an operation and a list as input and reports the combination of all the elements in the inputted list using the inputted operation. For example, the expression below reports: 3 + 1 + 1 + 0 + - 1.

Read this out loud as "combine the elements of the list {5, 6, 2, 3} with addition."

Combine is a higher order function; it is a function that takes a function as input. You've seen several higher order functions already: for each (in Unit 2 Lab 2), keep (in Unit 2 Lab 3), and map (in Unit 3 Lab 1).
