Op Art Hints

Below is a possible algorithm for the program. Compare with your own algorithm.

when green flag clicked://drawing functionality is active for the mouse
		hide the arrow sprite
		set pen color and size
		initialize variables drawingDone=false, rectangleList=empty list
		repeat until drawingDone
			if mouse down 
				set corner1=mouseX,mouseY
				repeat until mouse not down
					clear
					set corner2=mouseX,mouseY
					drawAllRectangles
				add the last rectangle with corner1,corner2 to rectangleList

The following blocks may help:

drawAllRectangles:
	clear
	draw the current rectangle as defined by corner1 and corner2
	draw all the rectangles in the rectangleList
