Selecting Specific Data

Purpose (do/learn; why this/here): Create a "Find Contact" button that uses a find contacts block to select items from the contact list based on the contact's name. Students get practice with list processing, navigating abstract data types, and using the higher order function, keep.

On this page, you will develop a "Find Contact" button to locate a particular contact in the list.

  1. If it isn't open already, open your U3L1-ContactList project.
  2. Write a block that takes a text string as input and reports either a list of the contacts it finds with that data or "not found" if the string is not in the contact list.
    Use these blocks:
    1. , which lets you find out whether the first string (for example, a contact's name in the list) contains the second string (for example, a name or part of a name).
      2. You learned about keep in Unit 2.
    2. , which filters the list for only the items that make the function true.
    3. , which returns true if the list is empty and false otherwise.
    4. Selectors and .
  3. Develop a "Find Contact" button.
    1. Give the sprite an appropriate costume.
    2. Make the script ask the user whose contact to search for and have the sprite say each of the matching contacts.
  4. Debug. Make sure everything works the way you want it to before going on.
  1. Develop a "Delete Contact" button.
    1. Create a new sprite with an appropriate name and costume.
    2. Make the script ask the user "Which contact do you want to delete?" and then use your find block together with keep to replace the contact list with a list of all contacts whose name does not match the inputted name.
    3. Make the script ask the user for confirmation (so the user won't accidentally delete their contact).
      4. For example, after the user enters the contact name they want to delete, the program could report that whole contact back to the user and ask if they are sure they what to delete it.
  2. Modify your "Delete Contact" script so that it lets the user choose which contact they want to delete when find reports more than one.
