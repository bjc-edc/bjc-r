On this page, you'll use abstraction to create a "Sentence Builder" project that builds complicated sentences like "the little elephant runs happily around the wise pizza" by joining different parts of speech: nouns, verbs, adjectives, articles, possessive pronouns, adverbs, prepositions, etc.

Recall, abstraction is hiding the complexity of a program by writing procedures to do specific parts of the task (like writing a program to draw one tree and then calling that program multiple times to draw a forest).
