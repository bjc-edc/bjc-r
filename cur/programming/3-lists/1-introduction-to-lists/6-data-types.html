A data type is a label that specifies what type of data a variable can hold. In some programming languages, there are different data types for numbers than for words—or even for different kinds of numbers (for instance, integers vs. decimals). In Snap!, there are variables (which each hold one piece of data that can be either a number or text) and lists (which hold multiple pieces of data—numbers or text). We can also create our own data types to help clarify our code.
