Selection Sort

One commonly used sorting algorithm is called selection sort. Here's how it works:

  • Step 1. If the input list is empty, there's no work to do.
  • The earliest item might be the first name in alphabetical order, or the smallest number in a list.
    Step 2. In an unsorted list, select the item that should come earliest in the sorted result. (If there are no remaining items, you're done.)
  • Step 3. Find the selected item from the list and place it in the first position of a new result list to be reported.
  • Step 4. Use a selection sort to put the remaining items in order as the rest of the result list.

(Don't take the word "step" too literally; you'll probably see when you write the code that at least steps 3-4 are in a single report block.)
