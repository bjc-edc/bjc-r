Selection Sort

One commonly used sorting algorithm is called selection sort. Here's how it works:

  • The earliest item might be the first name in alphabetical order, or the smallest number in a list.
    Step 1. In an unsorted list, find a way to select the item that should come earliest in the sorted result. (If there are no remaining items, you're done.)
  • Step 2. Pull the selected item out of the list and place it in the first position of a new result list.
  • Step 3. Use a selection sort to put the remaining items in order as the rest of the result list.
