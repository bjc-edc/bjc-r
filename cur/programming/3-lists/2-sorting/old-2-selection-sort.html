Selection Sort

On this page you'll learn one of many possible algorithms for sorting the items of a list.
  1. As a class or in small groups, see if you can invent an algorithm (just in words, not Snap! code) to sort a list.
There are many different sorting algorithms. In Unit 5, you'll go into algorithms in more depth, and learn several of them. But for now, you'll use one of the simplest, called selection sort.

We want to sort the contact list by name, but we're going to start with a simpler problem: sorting a list of numbers.
