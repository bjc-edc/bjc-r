A Tool for Creating Bar Graphs

In this project, you will create a data visualization tool general enough to generate bar graphs for a multitude of data sets. You will deepen your understanding of abstraction and gain practice with Higher Order Functions such as map.

  1. Open this starter project. You are given some blocks. Note that the first block is a constructor of a new ADT (Abstract Data Type) called data with two fields label and value. The second and the third blocks are the selectors for the new ADT that extract label and value given a data input. The next block finds the largest element of a list. Explore them and understand how they work.
