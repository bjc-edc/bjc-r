En esta página, desarrollará un tipo de datos abstractos (ADT) para almacenar y acceder a fechas, y adaptará su ADT de contacto para usarlo para almacenar el cumpleaños de cada contacto..

A medida que agregamos más complejidad a los contactos (cumpleaños, dirección del trabajo, etc.), la ubicación de cada punto de datos (como el número de teléfono) dentro de un contacto puede cambiar. Aquí es donde ayuda un tipo de datos abstracto. Si tuviera que mover el número de teléfono a medida que refina su ADT de contacto, podría arreglar cada bloque que usa el número de teléfono, o si tiene un ADT, podría simplemente arreglar el único selector que elige el número de teléfono de la lista. Con ese único cambio, todos los demás bloques que usen ese selector obtendrán los datos que necesitan automáticamente.

Observe que el nombre, la dirección y el texto son cadenas, pero la fecha de nacimiento es una lista. Puede combinar elementos de diferentes tipos en una lista.

  1. Cree un ADT para administrar tres puntos de datos para cada fecha: mes, día y año.
    1. Cree un fecha constructor que acepte estos datos como entrada y luego informe una fecha como una lista de tres elementos.
    2. Escriba tres bloques selector para recuperar los datos específicos (mes, día o año) de la fecha ingresada.
    3. Depure cualquier problema con el constructor o los selectores.
  2. Integre su ADT de date en el ADT de contact.
    • Edite la definición del bloque del constructor contacto para aceptar una fecha de nacimiento como una lista usando el constructor fecha.
    • Cree un fecha de nacimiento del contacto selector para recuperar la fecha de nacimiento del contacto ingresado.
    • Adapte su botón "Agregar contacto" para ingresar este nuevo tipo de contacto en su lista.

      Adición de contactos con cumpleaños

      Puede usar varios bloques de preguntas y respuestas junto con variables de script.
