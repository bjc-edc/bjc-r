Contact List App

All cell phones have some kind of Contacts list. In this lab, you will build a Contacts app yourself.
  1. Load this starter version of the app.
  2. Press "Add Contact" and read the script to understand how it works. Each new contact is a list, so the list of contacts is a list of lists.
  3. Fix the block so that it works similarly to.
  4. Clear your contact list and add a few made-up people to make sure everything works.
  5. Write a reporter that takes any part of a person's name as input, and reports that person's phone number, or "not found" if the name is not in the contact list. Here are two new blocks to help you:
    1. The block will help you find out whether a particular contact's name matches the user's input string.
    2. The block will apply a test (its first input) to each item of a list (its second input), and will report a new list containing only those items for which the test succeeds. (Remember that when you have a lot of contacts there might be more than one named John. If that happens, just give the phone number of the first one in the list.) If none of the items satisfies the test, then keep will report an empty list.
  6. When someone calls your phone, your phone service sends the calling number to the phone. But if the number is in your contact list, what the phone actually displays is the contact's name. So you'll also want a block that searches for a phone number and reports the contact's name.
  7. Our starter version of the app represents a person's name as a single text string containing all parts of the name. This is fine for displaying the name on the screen. But your phone's contacts app can also display an alphabetical list of all the contacts sorted by last name. So we really want to keep the last name separate from the first (and middle if any) name.
    1. Modify the block so that it asks separately for first-and-middle name and for last name, and reports a list of the two answers. (In some cultures, the family name—the one you want to use for alphabetizing—is written first, For example, in "Sun Yat-Sen" the family name is "Sun." So instead of asking for "first and middle name(s)" and for "last name" you might want to use the terminology "given name(s)" and "family name.") Make sure that adding a new contact does what you expect.
