The For Each Block

What students do: They build a block that takes a list of points as input and connects the dots. Useful for drawing and, later, for plotting data points.
What students learn: They think about problem decomposition in the form of tool-building, and creating and debugging an algorithm. First they build a block that goes to one point, using item of to decompose a pair of coordinates into the x and y components. Then they build a script that uses for each to go to all points, then a block that is defined by that script. In each case, they focus on the nature and use of the input.

Follow-up: I think that after this, students should use the thinking that they've done in problem 5. They should have the experience of clicking on some data points on an imported line graph (perhaps from a news article), reading the coordinates, making a list of them, and having Snap plot that line graph.
