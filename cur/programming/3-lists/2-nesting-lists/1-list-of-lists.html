Contact List App

Lists can contain other lists—all of the same size—to make tables (arrays) of information. You will create a Contact List app

  1. "U3L2-ContactList"
  2. called contact list to store the information.
  3. Initialize contact list as an empty list: .
  4. Create buttons for "Add Contact", "Show Contact", and "Delete Contact" either by downloading and importing the images below into the sprite's "Costumes" tab or by drawing buttons of your own.
  5. Write a script for "Add Contact" that, when clicked:
    • Asks the user for a phone number
    • Adds that number into a new contact in the contact list
    • Requests and stores the first name and last name of the contact
    6. You'll want to add some data as your test out your buttons, but don't spend time adding contacts yet.
  6. Create a "Show Contact" button that, when clicked:
    • Asks which contact to display
    • Searches for that contact
    • If contact exists, displays the selected contact
  7. Create a "Delete Contact" button that, when clicked:
    • Asks which contact to delete
    • Searches for that contact
    • If contact exists, asks for confirmation
    • If confirmed, deletes the selected contact
