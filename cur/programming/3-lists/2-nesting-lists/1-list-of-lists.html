Make a Contacts List App

make a contacts list app, in which each entry (abstraction!) is a contact. If we include address, then U4 can locate the contacts on a map and U5 can sort it. Let the U3 version of the project search the list for a contact and display it. It would be nice if it also could correct/update the information in a contact or delete a particular contact (done on this page, 1).
I think Paul's idea was to lead with the project. Finger exercises can come later. What makes it different is the introduction of an abstract data type for contacts. So I'm proposing to take Paul's version as a starting point, and have students improve it. --bh
