GPS Data

A computer forensic specialist who works for a law enforcement agency might analyze GPS (global positioning system) data.

You have a set of GPS coordinates from a device the suspect used to communicate during the time of the crime. Use them to reconstruct the movements of the perpetrator.

Locations in this format are called Decimal Degrees:
DDD.DDDDD° (latitude), DDD.DDDDD° (longitude)
If you search for "37.873808, -122.254173" in Google Maps, you can see the corresponding location, near the Greek Theater on the campus of U.C. Berkeley.

It's not necessary, but if you are interested, read the following quick introduction on how GPS coordinates work: E-How Tutorial on GPS coordinates.

The data given to you are 10 GPS coordinates, which can be used to determine a location on the Earth. Each row is a separate coordinate, consisting of two numbers, a latitude followed by a longitude.

Most computer based mapping systems display this format for GPS coordinates. Positive values of latitude are north of the equator, negative values to the south. Most programs (not all) use negative values for west longitude.

Transform the data

Transform these GPS coordinates into coordinates that will display nicely on this x-y grid. These problems will go through one way to do this (there's more than one way).
    Yes, subtracting -122.27 is the same as adding 122.27.
  1. Every x-coordinate starts with "37.86" and every y-coordinate begins with "-122.27". This information is not helping us tell the points apart. Subtract 37.86 from every x-coordinate and subtract -122.27 from every y-coordinate.
  2. Center the coordinates so they can be visualized in an x/y-coordinate system. Data can be easily centered by subtracting the mean of the data from all data points.
