  1. Load your grapher from before; it should have been saved as "U3L4-GraphingApp."
  2. Look in the Variables palette for this dataset on a classroom of 27 sixth-graders: . Each data point in this set represents a person and contains four items:
    • age in months
    • height in centimeters
    • weight in kilograms
    • gender
    Your was written to expect only two numbers in each datapoint as it was built with blocks like . Will it even work with this dataset? Try it. What does it do?
  3. Create a selector age which extracts the age of an input person.
  4. Similarly, create selectors height, weight, gender which extract the height, weight and gender of an input person.
  5. Now use the above selectors to create custom blocks that report:
    • ages—a list containing only the 27 ages
    • heights—a list containing only the 27 heights
    • weights—a list containing only the 27 weights
