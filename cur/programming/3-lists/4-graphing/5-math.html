The triangular number function adds all of the whole numbers from 1 through the input number. So triangular number (n) reports 1 + 2 + 3 + \cdots + (n - 1) + n.

The factorial function multiplies all of the whole numbers from 1 through the input number. In other words, (n) ! reports 1 \times 2 \times 3 \times ... \times (n - 1) \times n.

To develop these procedures, you will use to get a list of the numbers to add or multiple and use the combine block to add or multiply them all together. The block takes an operation and a list as input and reports the result of combining all the elements in the inputted list using the inputted operation. For example, the expression below reports: 0 + 1 + 2.

Combine is a higher order function; it is a function that takes a function as input. You've seen two higher order functions already: keep (in Unit 2 Lab 3) and map (in Unit 3 Lab 1).
