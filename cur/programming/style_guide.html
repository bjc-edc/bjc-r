  • References to text in the app (such as the "OK" button or the "Motion" palette) should be in quotes.

How to Make an Picture of Blocks

  1. Build the script you want in snap
  2. Choose "Zoom blocks" from the settings menu and enter "10"
  3. Scroll around to locate the script
  4. Right click the script and choose "script pic..."
  5. Save the file to the unit's new image directory, e.g. /bjc-r/img/3-lists

How to Link to a Script
