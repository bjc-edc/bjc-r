: Predicate and Boolean

A predicate is a procedure that asks a true/false question such as . Predicates report a Boolean value (either or ).

Predicates fit into a hexagonal input slots of conditionals, such as in and . Predicates can help your program decide when to do something.

Predicates can also be used with to decide which things to keep. The keep block takes a list and a predicate as input, and it reports a new list keeping only the items from the list that make the predicate true. Essential Idea

You write the predicate that does the checking, and keep applies that predicate to each item in the input list and then reports a new list of only the items that make your predicate true.

Note that the blank input slot in the predicate is required when using keep. This is where the item from the list goes each time the question (predicate) is asked.
