In computer science, a piece of text is sometimes referred to as a string. There are two custom substring blocks included in this project. They each report only a portion of a string:
  • The substring starting with block reports the part that starts with (and including) the text you enter in the input slot.
  • The substring up to block reports only the part up to (but not including) the text you enter in the input slot.

You can use these two blocks together to report a specific piece of text on the page.
