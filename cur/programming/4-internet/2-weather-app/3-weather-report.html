Building A Weather Report

Now you can query the National Weather Service for current temperature and weather conditions. Let's combine that with our work on lists.

Idea here: build two types of reports. One, a bunch of cities and their conditions. Two, a sentence report like "In Boston, it's rainy and 62 degrees." Note that in some cases the condition report is unsuitable for a sentence so there would have to be some switch-outs, if that's sensible.

  1. Identify some other Web sites where information in the URL is used in processing. For each site, describe how the information is used.

The National Weather Service is a U.S. Government agency providing current weather and forecasting. Like Google, the National Weather Service's website changes from different inputs, such as these:

http://forecast.weather.gov/zipcity.php?inputstring=10002

Here, inputstring is a ZIP code, a five-digit code matching a town or part of a city. Try entering your ZIP code.

http://forecast.weather.gov/zipcity.php?inputstring=Boston, MA

In this lab, you will pull the most useful bits of information from these pages to build a simple weather app.
