A Brief History of the Internet

Here is a brief timeline for the development of the Internet:

1969: Arpanet, first real network launched, UCLA and Stanford are first hosts
1971: First Email
1973: The first trans-Atlantic connection
1974: The beginning of TCP/IP
1977: The first PC modem
1978: First Spam
1982: The word “Internet” is used for the first time. The first emoticon :-)
1984: Domain Name System (DNS) established
1987: The Internet grows to 30,000 hosts
1988: First major malicious internet-based attack
1989: AOL launched
1991: CERN introduces the World Wide Web to the public. First web page created.
1991: Gopher, the first content-based search protocol, launched
1991: The first webcam
1993: Mosaic, the first graphical web browser for the general public, launched
1994: Netscape Navigator launched
1995: Commercialization of the internet (Amazon.com, Craigslist and eBay)
1998: Google launched
2000: The dot-com bubble bursts
2001: Wikipedia launched
2004: Facebook launched
2005: YouTube launched
2006: Twitter launched
2011: Twitter and Facebook play role in Arab spring
