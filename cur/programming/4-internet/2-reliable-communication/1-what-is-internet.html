What Is the Internet?

People talk as if "the Internet" and "the World Wide Web" are the same thing, but they are not. The World Wide Web is the collection of interlinked website documents (such as HTML files) that you can view with a web browser by typing URLs such as http://bjc.berkeley.edu/website/privacy.html.

What is a URL?

But the Internet is more general than the World Wide Web. It also supports email, file transfers, mobile apps, texting, and many other ways that computers communicate behind the scenes.

The Internet is a massive network of computers that communicate. They do this primarily by using a pair of protocols (standards for communication):

  • IP (Internet Protocol), an addressing system that finds paths to distant computers
  • TCP (Transmission Control Protocol), a transmission system that assures reliable transmission of data

Given the scale of the Internet, the computing algorithms to ensure that all data reach their destinations must continue to work even as the Internet grows. To meet this need, the Internet was designed to be:

  • Redundant: There are multiple pathways for data to travel; so even if one pathway is unavailable, there is still another way to go.
  • Hierarchical: The Internet addressing systems obey a hierarchy to improve efficiency. This hierarchy works much like the postal system, which locates people first by state, then city, then street, then house number, then apartment number, and then finally by person.
