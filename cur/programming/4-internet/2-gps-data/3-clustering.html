Based on the visualization, Big Burgers will place their restaurant in the zone with the largest number of customers. But, which one is it?

Taking a large data set and breaking it into smaller groups is called cluster analysis. Visualizations of data are especially helpful in identifying and classifying clusters.

  1. Given a specific pair of GPS coordinates, how might you decide which of the three zones it is in?
  2. Create a zone block. The input is a pair of GPS coordinates, and the output indicates which zone the coordinates are in.
  3. Use the zone block and the full set of GPS data to determine which of the three zones has the most customers.
