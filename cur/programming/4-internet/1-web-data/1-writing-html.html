Most web pages are written with HTML (hypertext markup language) and read with HTTP (hypertext transfer protocol).

  1. Watch this video from code.org on HTTP and HTML:
  2. Describe HTTP and HTML in your own words.

There are many sites on the internet that can help you learn to code. w3schools.com is another example.

  1. Try out the w3schoools.com HTML example. Experiment with the code and watch the impact on the result:
    1. Create a second heading and more paragraphs.
    2. Try deleting some or part of the html tags like "<title>" or "</h1>".
  2. Add to your written assignment more detail about what HTML can do.
  3. Try out the CSS example. Make some changes to the code:
    1. Notice the two different ways color is coded (as a hexadecimal number: "d0e4fe" and as a word: "orange"). Experiment with style of the body, level 1 heading, and paragraph.
    2. Experiment with removing some of the CSS syntax (like the brackets, the colon, or the semicolon).
  4. Add to your written assignment more detail about what CSS can do.
    5. Syntax is like grammar for a programming language. It defines what words and symbols can go where. Snap! is block-based lanague, so you won't deal with syntax for most of this course.

    You can "View Source" for any web page using your browser. You might be able to find it in a menu, or look it up.

  5. Look at the source code for a few different web pages and make a list of all the HTML formatting commands you can figure out. For example, what code makes text appear in bold or italics?
