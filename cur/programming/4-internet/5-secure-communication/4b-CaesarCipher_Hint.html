Caesar Cipher Hints:


Here are some possible steps in the implementation of the Caesar Cipher:
  1. Create and initialize variables that will store the input text, shift value and the output text.

  2. Ask the user to enter message to be message to be encoded. Set the inputText variable to the input from the user.

  3. Ask the user for the shift value of the Caesar Cipher. Check that user input is a number.

  4. Set the shiftValue variable to this input. Ask the user if they wish to encode or decode and invoke a custom procedure to do the encoding or decoding.
