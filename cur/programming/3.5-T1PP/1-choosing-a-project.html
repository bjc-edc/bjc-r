This project will give you a chance to use the programming concepts you've been learning to build a project that you chose or design from scratch. It will also help to prepare you for the Advanced Placement Create Task later in the year.

You will do each of the following steps of the Term 1 Programming Project with a partner. On the actual AP Create Task, you will also go through very similar steps individually.

  • Choosing and developing a Snap! project to develop as a team
  • Submitting a PDF of your code and a video of your project
  • Writing a team reflection on your programming project
