This project will give you a chance to use the programming concepts you've been learning to build a project that you chose or design from scratch. It will also help to prepare you for the Advanced Placement Create Task later in the year.

You will do part of the project with a partner:

  • Choosing and developing a Snap! program as a team
  • Submitting a PDF of your code and a video of it running
  • Writing a reflection on your team programming project

You will do part of the project by yourself:

  • Creating a second Snap! program by yourself or designing a new feature for your team program that you develop by yourself
  • Submitting a PDF of your code and a video of it running
  • Writing a reflection on your individual programming project
