    • Report whether or not (using true/false) the inputted word has the inputted number of letters.

    4. Specifying a Variable's Input Type

    You may have noticed that input slots can have different shapes. This is because they accept different types of input. To change the input type:

    • Click the variable you want to edit from inside the Block Editor. In this case, click numberOfLetters.
    • Click the right triangle in the Edit Input Name pop-up box.
    • Choose the input type that you want. For numberOfLetters, select Number. Then click OK on both the Edit Input Name and Block Editor windows.
    The shape of the variable's input slot tells people what type of input the block expects for that variable.
