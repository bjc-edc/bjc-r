Solving a Word Puzzle

On this page, you will create predicates that you will combine using the and block and use inside keep to find words with specific characteristics.

  1. Open this project and run set WORDS LIST to initialize the words variable.
    2. Look inside. The set WORDS LIST block uses the http and split by blocks to access a text file online and then makes each line in the data an item in a list called words.
  2. Use the block in the Operators palette together with = and keep to find words of a certain length in the words list.
    The green block is in the Operators palette. (Do not mistake it for the red block for lists.)
    1. Find all the words that have 8 letters.
    2. Find all the words that have 5 letters.
    3. Find all the words that have 11 letters.
  3. Create and test a predicate block to check a word for a certain number of letters. Examples and information about input types are below. Your block should:
    • Accept two inputs: a word and the number of letters it should have, and
    • Report whether or not (using true/false) the inputted word has the inputted number of letters.
  4. Use the block with = and keep to find words with letters in certain places.
    1. Find all the words that begin with the letter i.
    2. Find all the words whose fourth letter is r.
  5. Create and test a predicate block to check for letters in certain places. Examples are below. Your block should:
    • Accept three inputs: a letter number for the place in the word, a word, and a letter, and
    • Report whether or not the letter number letter number in the word is the inputted letter.
