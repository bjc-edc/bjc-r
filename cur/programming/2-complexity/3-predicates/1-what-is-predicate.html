As you know, predicates are reporter blocks (functions) that always report a Boolean value (they report only the values or ). In Snap!, they are represented by hexagonal blocks. Predicates compute the condition used by conditionals (such as if or repeat until) to decide when to do something. Predicates ask a question such as "Is the random number 3?" or "Is this sprite touching the sprite called 'Sprite'?"
