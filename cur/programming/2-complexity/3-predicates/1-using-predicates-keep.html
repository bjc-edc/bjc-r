Predicates

In snap, predicates are hexagonal in shape, not oval like other reporters.

Notice that the two Booleans are themselves predicates because they only report true or false.

Predicates are reporters that always report a Boolean value (they always report either or ).

You have seen predicates before. Predicates are the condition used by conditionals (such as if or repeat until) to decide when to do something. They can also be used with keep to decide when to keep something.

The Keep Block

The block takes a list and a predicate (a true/false question) with a blank input slot as input, and it reports a new list keeping only the items from the list that match the condition defined by the predicate.

Like the for block, keep isn't installed automatically. It has already been added to the project you will open below, but in other projects, you'll need to import tools to access the keep block.
