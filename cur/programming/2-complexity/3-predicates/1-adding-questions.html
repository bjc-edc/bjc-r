Predicates

Predicates are hexagonal reporter blocks (functions) that always report a Boolean value (they only report either or ).

You have seen predicates before. Predicates are the condition used by conditionals (such as if or repeat until) to decide when to do something. Predicates ask a question such as "Does the player list contain the player's answer?" or "Is this sprite touching the sprite called 'Sprite'?"
