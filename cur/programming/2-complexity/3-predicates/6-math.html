In this project, you will create a collection of math tools building on the blocks you built in Lab 3 Page 1: What's a Predicate?.

    You learned how to export and import blocks in Unit 2 Lab 1 Page 4: Importing Greet Player into Another Program.

    If the imported even?block does not work, review the section about "Debugging Dependencies" on the Brick Wall page.

  1. Export all of the custom blocks from your U2L3-Predicates project. (These should include between?, divisible?, even?, and perhaps also integer? if you had time to build it.)
  2. "U2-MathTools"
  3. Import those blocks into your new project file, test them and resolve any issues, and save.
  4. Develop a number of divisors block, and use it to answer these questions:
    • What kinds of numbers have an odd number of divisors?
    • What kinds of numbers have exactly two divisors?
