Predicates can be used with conditionals to decide when to do something, and they can also be used with keep to decide when to keep something. The block takes a list and a predicate with a blank input slot as input, and it reports a new list keeping only the items from the list that match the condition described by the predicate.

You write the predicate that does the checking, and keep applies the predicate to each item in the input list and then reports the list of items that make your predicate true.

Notice that the blank input slot in the predicate is required when using keep. This is where the item from the list goes each time the question is asked again.
