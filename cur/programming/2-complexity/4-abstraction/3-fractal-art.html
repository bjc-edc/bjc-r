Fractal Art

In this project, you will learn how to nest repeat blocks inside repeat blocks to generate complex structures.
 

It is possible to nest one repeat block inside another. Since snap processes scripts from top to bottom, the first repeat will begin first, and the second repeat will have to complete all its repetitions before the first repeat continues.

Nesting means organizing information in layers or embedding objects inside other similar objects.

Using Abstraction to Nest Triangles

Does all this copying and pasting of code feel awkward to you? You know a better way: abstraction. In Unit 1 you used a pinwheel block to implement asterisk and polygon, rather than copying the code. Here, you can use a block to manage the abstraction too, but in this case, we want very similar code (a smaller triangle) repeated inside, so we will actually use the same block inside itself.

  1. Create a repeatedTriangle block.
    1. Design the block with two inputs, size and color.

    2. Use your first unnamed triangle script from problem 1 as a model. To have an input for the color, you'll have to use the numeric block instead of the color-picker block .
    3. Click the "Apply" button in the Block Editor so that the block appears in the palette on the left.
    4. Where you would have dragged in another copy of the script (between the move and turn blocks), drag in a repeatedTriangle block. The size input should be half the current value of size; the color input should be ten more than its current value.

      You are using repeatedTriangle in its own definition. When a block refers to itself, the block is said to be recursive. Recursion is one of the most powerful techniques in computer science and you will learn more about it in later projects.
  2. Try repeatedTriangle with a size of 100 and a color number of 0. It won't work as expected. Think through the steps in the code. Can you determine what is happening?
