Debugging Tip: Visible stepping in Snap!.

Snap! has a fantastic debugging tool called Visible stepping which will allow you to step through the blocks of your code one by one as each block is being executed, revealing the effect of each block. You can adjust the speed of this stepping and even bring it to a halt if you choose. If there are errors in your program, stepping will help you identify the block(s) causing the errors.
Click for a demo.

Below you see an animation about how to use Visible stepping in Snap!. Be sure to watch the code not just what the sprite does. Notice how it is possible to adjust the speed at which the stepping is done. During the stepping, the color of the block currently being executed changes to cyan. Notice how stepping is done when there is a loop within a loop.

In Visible stepping, if you bring the slider all the way to the left, then you can single-step your code by pressing the Pause/Play button next to the Green Flag.
