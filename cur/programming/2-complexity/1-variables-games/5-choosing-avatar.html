When you used item of before (for example, inside who and does what) , you were accessing elements of the list randomly. But you can also select a specific element by specifying its position.

: Index

The position number is called the index of the element in the list.

Lists can contain anything: letters or words, costumes, others lists, or even blocks. You can use my (costumes) to report a list of the costumes for your sprite.

And you can use item of together with my (costumes) to select a costume in a specific position in the list.

Some blocks (such as item of) have input slots that expect a list. You can tell because the input slot looks kind of like a list:
