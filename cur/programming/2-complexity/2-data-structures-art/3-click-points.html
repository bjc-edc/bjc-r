Click to Draw

On this page you'll make the list of points for a letter (or other shape) by writing a program to collect the points at which you click the mouse into a list, instead of typing point coordinates in yourself.
Alphie: Typing in all these coordinates is a pain. Why can't we just draw the letters on the screen, and then click on the endpoints of the lines?
Betsy: Umm, I think we'll get in trouble if we draw on the screen. But we could draw on tracing paper (so we can still see the mouse cursor) and hold the picture up to the screen.
Alphie: We'll have to write a block that keeps waiting for a mouse click, and then adds the mouse coordinates to a list. When all the points have been entered, the block should report the list of points.

  2. Finish writing list of points so that it collects points from mouse clicks as in the movie above and reports the list. You'd use it this way:
    1. Bear in mind that the purpose of this activity is for you to learn how to write code that collects points into a list. You're going to use that knowledge on the next page. So don't get hung up on the visual parts, which we've mostly provided for you.
    2. Start by reading the starter code and try to understand its structure. For example:
      • It uses a forever block. When, and how, does it report a value?
      • What is the if else block testing, and why?
      • Where you see , you're going to replace that block with other blocks as needed to implement the algorithm.
      • Can you use a script variable for the list inside list of points, or does it have to be global?
    3. Start with an empty list, and add a point for each mouse click.
    4. The stamp block is built into Snap!. It puts a picture of the sprite's costume on the stage at the sprite's location.
    5. Notice how convenient it is to have go to and draw blocks built to use the point abstract data type, instead of two number inputs per point. Once you've created a point from the mouse coordinates, you never have to think about coordinates again in this activity.
