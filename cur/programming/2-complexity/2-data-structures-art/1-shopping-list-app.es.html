Adición de Elementos a Una Lista

AAP-1.C.2

Un elemento es otro nombre para un artículo de una lista. (Si el mismo valor está en la lista dos veces, eso cuenta como dos elementos diferentes). Cada elemento tiene un índice (posición) único en la lista.

Puedes usar el bloque o para agregar elementos a una lista.
  • Insertar coloca el nuevo elemento antes del lugar que especificaste.
  • Agregar coloca el elemento después del último elemento existente.

AAP-1.D.6

Asignando una lista a una variable le permite usar un nombre para representar todos los elementos de una lista como una unidad.
