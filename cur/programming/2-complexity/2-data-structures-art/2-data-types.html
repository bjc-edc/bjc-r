An abstract data type (or ADT) is a kind of data that's meaningful to your program but not built into the language. You develop it as you program.

For example in this lab, you'll create a point data type with both to construct the ADT and and to select either coordinate of a point.

The word "abstract" is often used casually as if it means something harder to understand or more complex, but in computer science, its meaning is almost the opposite. ADTs are things that you, the programmer, create to make your program easier for you and others to read, debug, and improve.
