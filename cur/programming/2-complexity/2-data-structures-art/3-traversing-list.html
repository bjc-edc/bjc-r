: Traversing a List
AAP-2.O.2

Traversing a list means looking at each item of the list. For each is iterative (that is, it's repetitive, like for, which also traverses lists). But unlike for, for each traverses the list without using index numbers.

This language should be tightened up. For example, it's not clear whether for traverses a list. --MF, 4/15/19
