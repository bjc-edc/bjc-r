: Traversing a List
AAP-2.O.2

Traversing a list means looking at each item of the list. For each is iterative (that is, repetitive, like for), but for each traverses the list without using index numbers.

    PROCEDURE Mystery (numberList, targetNumber)
{
  counter ← 0
  FOR EACH number IN numberList
  {
    IF (number > targetNumber)
    {
      counter ← counter + 1
    }
  }
  RETURN (counter)
}
