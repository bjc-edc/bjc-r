Tic Tac Toe Project Overview


In this series of Tic Tac Toe projects, you will build three versions of the game, each version making the computer’s involvement in the game more intelligent than the version before:

Phase Video Description
Phase1 Basic Human versus Human game where the computer simply displays the X’s and O’s but the users themselves decide on a win or a draw. This version will be a lot like two players playing the game with paper and pencil. Tic Tac Toe Phase 1 example.
Phase2 Human versus Human game where the computer decides on a win or a draw. In this version, the computer starts gaining awareness of the game and begins to meaningfully contribute to the game. Tic Tac Toe Phase 2 example.
Phase3 Human versus Computer game where the computer not only decides on a win or a draw but it also has some AI (Artificial Intelligence) features that allow it to “think” about a proper next move and simulate the moves of an intelligent human opponent. Tic Tac Toe Phase 3 example.

Note: In the last version, even though the computer will be aware of the nuances of the game, it will not be unbeatable. As an extension activity, you will be encouraged to develop an AI that is unbeatable at Tic Tac Toe. Such AI, while fun to code, will not be so much fun to play against.
