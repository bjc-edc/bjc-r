  1. En este proyecto, utilizará rectángulos repetidamente, así que primero recuerde cómo crear un bloque rectángulo.
    Tenga en cuenta el uso de los bloques pluma abajo y pluma arriba dentro de la definición del rectángulo. Normalmente no hacemos esto, pero aquí se hace para simplificar el código más complejo que se avecina. Esto asegurará que una vez que se dibuje un rectángulo, la pluma esté levantada y no tenga que preocuparse por los garabatos no deseados mientras su sprite de dibujo continúa moviéndose para ejecutar el resto del código.
  2. Recuerda también el código pinwheel que escribiste en la Unidad 1.
  3. En este proyecto, reemplazará las ramas de molinete con rectángulos como se muestra a continuación. Tenga en cuenta que el uso del color es opcional y se usa aquí para mayor claridad.

    Ahora crea un bloque molinete con rectángulos. Tendrá dos entradas más ancho y largo para el ancho y largo de los rectángulos.

    Su código será idéntico al código molinete excepto que tiene el bloque rectángulo insertado entre los dos bloques mover que crearon las ramas del molinete.

  4. Ahora, use el menú "Variables" para crear variables que le permitirán variar todos los aspectos de los diseños que cree y no olvide inicializar estas variables justo después del bloque cuando se hizo clic en la bandera verde
  5. Puede convertir estas variables en controles deslizantes modificables haciendo clic con el botón derecho en el icono de la variable que se muestra en el escenario y eligiendo la opción "control deslizante". Y luego puede establecer los valores mínimo y máximo de un control deslizante como se muestra.

  6. Haga clic en la animación a continuación para ver un ejemplo de cómo puede usar la función de control deslizante en Snap! para generar y explorar una gran cantidad de variaciones de sus diseños.
  7. Su código debe asegurarse de que la computadora esté siempre en el reloj en caso de que el usuario cambie alguna de las variables a través de los controles deslizantes. Puedes hacerlo con un bloque siempre.
    El bloque deformación se asegura de que el molinete con rectángulos se dibuje todo a la vez en lugar de hacerlo paso a paso.
  8. Una vez que el código esté escrito, verifique que todo funcione según lo previsto. Es posible que deba colocar su dibujo a pesar de manera adecuada para aprovechar al máximo el escenario que tiene. Ahora mueva cada uno de sus controles deslizantes para generar y explorar una gran variedad de diseños inspirados en el arte islámico.
