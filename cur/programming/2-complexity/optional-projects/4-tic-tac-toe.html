Tic-Tac-Toe Using Sprites


In this yearlong project, you will begin developing a program that plays Tic-Tac-Toe as well as you do.

On this first page, you display a Tic-Tac-Toe board and let two human players enter moves.

  1. Open this starter project. A sprite has already been created for you with three costumes: Empty, X, and O.
  2. You will be using the clone feature in Snap! to create the 3×3 Tic-Tac-Toe board. A clone is just a copy of an existing sprite. In the starter project is this demonstration script:

    Click on the sprite on the stage and watch what happens.
    • Note that it is the original sprite that moves to a new random position after cloning itself.
    • Note that both copies are active sprites. Both are clickable (try it) and draggable (try that too).
    • The create a clone block takes an input because it can copy any sprite, but in this project there's only one sprite so you should use myself as the input.
Don't go on until you're sure you understand cloning. Discuss it with others as needed.

The block is called delete this clone rather than delete this sprite because it works only for clones. So when you click the green flag, the original sprite and all its clones will run the script, but only the clones will be deleted. There will then be one sprite left on the stage.

  1. Delete the when I am clicked script. You'll make your own different one soon. Also, at the bottom of the scripting area, there's an additional when flag clicked block above but separate from the script just below it. Click them together.
    • The wait block is needed to ensure that the other when flag clicked script has time to delete old clones before we make new ones.
    • makeBoard is an empty script. You're going to write it in the next problem.
    • The variable X's turn? will be true when it's X's turn to play, or false when it's O's turn. When alternating between two values, it's convenient to use true and false because you can use to switch between them.
  2. Edit the block makeBoard to set up nine clones in three rows of three:

    Hint:

    • The costumes are all 50 steps tall and wide.
    • Make sure the clones start out with the empty square costume.
    • After cloning nine times to create the board, hide the original sprite so it does not interfere with the game as a tenth square. Be sure to show the original sprite before cloning it, so that the clones will be visible.
  3. Now write a when I am clicked script that will allow each clone to put on the proper costume when clicked: X or O depending on whose turn it is. Also think, how should a square behave when it is clicked while already wearing an X or O costume.
