Siri y otros robots están programados para generar lenguaje. Siri habla; algunos robots escriben. Has desarrollado un programa, chisme, que escribe. En este proyecto, ampliarás las habilidades de la computadora con el lenguaje mediante el desarrollo de un bloque reportero, plural, que toma un sustantivo como entrada, deletrea correctamente y genera el plural.

Deletrear un plural puede parecer casi automático para una persona, pero no es simple: una computadora necesita un algoritmo para hacer esa tarea.
Este proyecto utiliza bloques ya cargados de la biblioteca "Palabras, frases" de Snap! Puedes cargar una biblioteca desde el menú Archivo Snap! en "Bibliotecas...".
