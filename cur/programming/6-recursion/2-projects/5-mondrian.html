Recursive Mondrian

Recursion is not just for fractals...

  1. Do a quick search for the Dutch painter Mondrian.

Mondrian's paintings were not created recursively, but we can use recursion to create similar images:

  1. Load this project. It has several blocks already created and the script that calls Mondrian already set up.
  2. Begin by programming the code for your base case, the rectangle block. It will need to draw a rectangle with a black border and use a very thick line of color to paint the inside. Be sure to preserve state transparency (put the sprite back where it started).
    The rectangle block already contains two custom blocks, set pen to black line, for the borders, and set pen to paint, which runs a random set pen color block from the colors list variable according to the thickness you determine. You can look inside these two custom blocks to see how they work.
  3. Click "r" a few times to be sure you have rectangle set up correctly. Each time, you should get a big rectangle with one of the five colors inside and a black border, like this:

  4. Then, program the code for the recursive case. The Mondrian block already contains some structure. Your job is to fill in all the blanks. Before you get started, think carefully about how the script is going to work.

    • You've already seen level 1. For level 2, Mondrian will randomly divide the rectangle into two pieces and draw a rectangle on each side. For level 3, Mondrian will randomly divide the rectangle into two pieces, randomly divide each of those two pieces into two pieces each, and then draw a rectangle in each of those four places. Level 4 will create eight rectangles, level 5 will create sixteen, and so on.
    • The percent variable gives a random percentage for how to divide up the rectangle each time. You'll have to determine how to calculate the lengths of the rectangles. (For example, if we are dividing a width of 200 with a percentage of 40%, we'll have a rectangle of width 80 and one of 120, and their heights will be the same).
    • The algorithm will be slightly different based on whether a rectangle is being divided up along the width or along the height. Notice that the if-else determines whether width or height is greater. Divide each rectangle along the longer side.

  5. Click "space" a few times. When you have Mondrian working, it should look something like this:
