Subsets Hints

  1. How many subsets does the empty set {} have?
  2. What should subsets report in the base case? This is a hard part to get right!
  3. How many subsets of {Apple, Orange, Banana, Pretzel} contain Pretzel? How many don't?
  4. Is there another way to describe "the subsets of {Apple, Orange, Banana, Pretzel} that don't contain Pretzel"?
  5. In the Snap! file menu, choose "Libraries..." at the bottom, then choose "List Utilities." You'll find the append block in that library helpful.
