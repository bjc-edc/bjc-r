Subsets

Given a set of things, a subset contains zero or more of its elements, without duplicates. Like ice cream bowls, order doesn't matter.

{Banana, Apple} does not count as a different subset from {Apple, Banana}.
For example, given the set {Apple, Orange, Banana} there are one-element subsets like {Banana} and two-element subsets like {Apple, Banana}. The original set {Apple, Orange, Banana} counts, and so does the empty set {} with zero elements.

  1. Write down all the subsets of {Apple, Orange, Banana}. How many are there?
  2. Write down all the subsets of {Apple, Orange, Banana, Pretzel}. Try to do this as simply as you can.

The subsets block takes a list as input, and reports a list of lists in which each item is a subset of the input list.

It doesn't matter in what order the subsets appear in the output list, but each subset must appear exactly once.
