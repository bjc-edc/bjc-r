These could be numbers instead of names, or anything that can be compared and ordered. Numbers and words often need to be sorted.

  1. Invent an algorithm for sorting the names. Try to describe the algorithm in enough detail so that someone else could use your description to sort a list.
  2. Now build your algorithm: a sort block that inputs a list and outputs a new list in alphabetical order. Test your algorithm using both names and numbers.
Comparing numbers and strings can be done using the same blocks, so your sorting algorithm should work equally well for names and numbers.
  1. Share your sorting algorithms. Describe some similarities and differences between different sorting algorithms.
