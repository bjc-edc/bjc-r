Writing Recursive Reporters

To write a recursive reporter, it helps to describe how the output is related to a simpler output of the same reporter.

The factorial function, n!, is defined to be the product of the integers from 1 to n.

  1. Say 9! as "nine factorial", not NINE!
    The value of 9! is 362,880. Find the value of 10!.
  2. Describe the relationship between 37! and 36!.
  3. Use these ideas to build a factorial block using a recursive reporter. Don't forget Betty's advice: base cases are important!

Word processing programs count the number of words and letters in documents. One way to count the number of letters in a phrase uses a recursive reporter.

  1. The phrase "in the course of human events" has 24 letters. How many letters are there in the phrase "When in the course of human events"?
  2. How does the number of letters in a 37-word sentence compare to the number of letters in its last 36 words?
  3. Recursion needs a base case. What sort of base case might there be for this situation?
  4. Create a letter count block that takes a list of words as input and reports the total number of letters in the words. The all but first of block may be useful, but there is more than one way to do this.
