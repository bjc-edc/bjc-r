Pair Programming Swap

Professional programmers use a technique called pair programming—writing computer programs in teams of two. One partner controls the computer, the other helps plan and helps catch errors. They switch roles often.

You will do that in this course, too, aiming for equal time in each role.

  1. If you haven't swapped yet, swap now.
