Getting Started

Experiment with these two blocks, using short input lists, to see what they do:

Discuss these questions with your partner:

  1. What's the biggest difference between the two blocks?
  2. What are some similarities between them?
  3. What does all but first of do?
  4. Fill in the empty parts of the following definition so that the block reports containing all the whole numbers from a to b. Note that this question does not involve a list.

  5. Now rewrite the block so that it reports a list of numbers instead of a sentence. Experiment with possible list combiners to use instead of join words.
