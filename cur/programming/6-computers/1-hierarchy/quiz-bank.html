Creating the negative of an image, where light areas appear dark, and colors are reversed
Correct. This transformation is reversible and is an example of a lossless transformation.
Blurring the edges of an image.

(Binary Sequences) A computer program uses 4 bits to represent integers. When the program adds the decimal (base 10) numbers 9 and 7, the result is 0. Which of the following is the best explanation for the result?
The result was affected by lossy data compression.
There is no compression involved here.
A round-off error occurred.
There is no rounding done here.
An overflow error occurred.
Correct. 9+7=16, which is beyond the capacity of 4 bits to express. 0=(0000)2 is the first and 15=(1111)2 is the last integer expressible in a 4-bit system. 16=(10000)2 leads to an overflow.
The result was approximated by a floating-point representation.
0 is not an approximation of 16.
