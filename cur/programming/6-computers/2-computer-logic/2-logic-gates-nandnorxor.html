Logic Gates: NAND, NOR, XOR

NAND-gate: not AND


A point is in A NAND B if and only if it is not in A AND B (intersection).

The following electric circuit may give you an idea how it is possible to emulate the Boolean operator NAND electronically.

The light bulb is ON only if it is not the case that both of the switches are ON.

Daily Life Application: A mechanism for a safe that releases a lock only when both of two keys are used.

NOR-gate: not OR


A point is in A NOR B if and only if it is not in A OR B (union).

The following electric circuit may give you an idea how it is possible to emulate the Boolean operator NOR electronically.

The light bulb is ON only if it is not the case that either or both of the switches are ON.

Daily Life Application: A mechanism for a safe that releases a lock only when either one (or both) of two keys are used.

Universal Gates: NAND, NOR

NAND and NOR gates are called universal gates as each alone can emulate NOT, AND, and OR.

XOR-gate: Exclusive OR


A XOR B is logically equivalent to either of these:
  • A XOR B = (A OR B) AND (NOT (A AND B))
  • A XOR B = (A AND NOT B) OR (NOT A AND B)
A point is in A XOR B if and only if it is in A or in B but not in both.

Daily Life Application: A mechanism that allows access to a resource when either of two users but not both attempts to unlock. A door that swings in either direction.
