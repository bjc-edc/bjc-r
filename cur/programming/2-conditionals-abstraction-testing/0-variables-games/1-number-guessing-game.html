Developing a Number Guessing Game with Script Variables

In this lab, you will use two kinds of variables to manage data in your games:

  • Script variables (also known as local variables) that are for use in a single script, and
  • Global variables that can be accessed anywhere in the app.

On this page, you will develop a number guessing game that uses a script variable to keep track of a secret number.

Using a Script Variable

You have also used variables that the for block gave you:

Both input variables and for variables belong to the script in which they're created; you can't drag one into a different script (or rather, you can, but it won't work). They are kinds of script variables.

A script variable usually exists only while the script is running.
Sometimes you will need additional script variables to hold information temporarily, while your script is running. The script variables block lets you do that.

For example, imagine a number-guessing game in which the player tries to guess the computer's secret number. The computer needs to remember the secret number so that it can compare the player's guesses to it, but once the game is over, the secret number does not need to be remembered. If you play the game again, you want to get a new secret number.

  1. "U2L1-NumberGuessing"
  2. Create a new command block called Number guessing game. (You might choose to put it in the Looks palette since there will be a lot of conversation using the say block.)
  3. Inside this new block, create a script variable called secret number and set it to a random number between 1 and 10.

Checking the Player's Guess

The computer should ask the player to guess the secret number again and again until the player gets it right. You will use the repeat until block, which is a loop (just like repeat, forever, and for), but it only repeats until a certain condition is met. In this case, it should repeat until the player's answer is the secret number.

: Boolean.
It's capitalized because it's named after a person, George Boole, who invented the branch of mathematics dealing with Boolean operations.

Repeat until is a conditional block, like if, as well as being a loop block. Conditionals make choices in a program based on a true-or-false condition called a Boolean value.
