The game is not that cool yet. You will improve it to make it seem more like a person is playing with you.

Here are six ways to make the code act more like a human. If you think of anything else that would make it even better, feel free to add it in. Try each of these in order. When you are done, your finished code should be able to deal with all of the challenges.

  1. The sprite should welcome the player and ask the player's name before beginning.
  2. If the player guesses incorrectly, the sprite should tell say so, using their name. For example, if Colleen is playing the game, the sprite should say something like Nope, Colleen, that's not my number. Try again.
  3. The sprite should also tell the player if the secret number is bigger or smaller than the number that the player guessed.
  4. Right now, the script always picks a number between 1 and 10. Change this so that it picks a number between 1 and a variable named maximum. At the start of the game, ask the plaer what that maximum number should be.
  5. Keep track of how many guesses it takes before the player guesses the right number.
  6. When the player guesses the secret number, say how many guesses it took, and congratulate them using their name.
