The script checks "forever" to see if is true. If it is, the sprite follows the instruction contained inside the if. If is not , the sprite is not instructed to move. Blocks, like if, that specify the conditions that control when some code is to run are called conditionals.

Blocks, like greater than, that report true when a condition is met and false when it is not met are called "predicates."

Predicate functions are like questions that can be answered with true or false. The two special values are represented in Snap! by the blocks and . They are called Boolean values, named after the great logician George Boole. Some predicates, like turbo mode?, will not yet make sense, but you are likely to be able to make a good guess about many others.
