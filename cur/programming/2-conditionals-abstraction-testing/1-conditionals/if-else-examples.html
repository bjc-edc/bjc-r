Blocks Using If and If-else

  1. Make a block that when given a light color, makes the sprite say the appropriate action. For example should say "go!"
  2. Make a block that takes in a percentage and makes the sprite say the associated letter grade. For example, might say "C."
  3. Make a block that takes in a temperature and says whether water will be liquid, solid, or gas.
  1. Edit your traffic signal block to change the sprite's costume to show the correct color light. You can find a set of stoplight costumes here.
