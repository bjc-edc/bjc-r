    The block reports the remainder when the first input is divided by the second (and we are using only positive integers). For example, because when 17 is divided by 5, the remainder is 2. When one number divides another evenly, the remainder is 0.

    In mathematics, we write: 17 (mod 5) ≡ 2 and 15 (mod 5) ≡ 0.

In problem 1, we needed several if statements because we were testing for several conditions. In problem 2, we need only one if because all integers are either even or odd. If the boolean expression (n mod 2 = 0) evaluates to false, we know that the number is not even, so it must be odd.
