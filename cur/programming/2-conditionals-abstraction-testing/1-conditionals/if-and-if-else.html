    The block reports the remainder when the first input is divided by the second (and we are using only positive integers). For example, 14 mod 3 reports 2 because 14 ÷ 3 leaves a remainder of 2. When a number is divisible by another, the remainder is 0.

In that second example, we didn't need to have an additional "if," because all whole numbers are either even or odd. If the boolean expression (n mod 2 = 0) evaluates to false, we know that the number is not even, so it must be odd.
