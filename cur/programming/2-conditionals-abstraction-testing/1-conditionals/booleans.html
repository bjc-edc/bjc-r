The if block sets a condition. The script checks continuously to see if that condition is true, and only if it is does the sprite follow the instruction . If is , the sprite is not instructed to move.

Computer scientists refer to blocks like if as conditionals. Blocks that report true when a condition is met and false when it is not met are called "predicates." These two special values, true and false, are called Boolean values (named after the great logician George Boole). These values are represented in snap by the blocks and .
