Reporter blocks can report many kinds of values. The plus block, for example, reports a number.

Some reporter blocks, like greater than report only two special values: when a condition is met and and when it is not met. Reporters that report only true and false are called "predicates."

In snap, these functions have the hexagonal shape: .

equals, not, and, and or are other predicates you will often find useful:

Predicate functions are like questions that can be answered with true or false. These two special values are called Boolean values, named after the logician George Boole.
