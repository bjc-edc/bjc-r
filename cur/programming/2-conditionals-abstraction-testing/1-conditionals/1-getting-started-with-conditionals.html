Conditional statements: Code like if and if-else and repeat-until allows your programs to perform different actions or computations depending on a condition that you specify. In computer science, we refer to such code as conditional statements.

 

Predicates: In this lab, you tested conditions such as x position < 0 and direction = 30 OR direction = 60. If those tests showed the condition to be true, the code behaved one way; if false, the code behaved a different way. Tests that report only true or false are called predicates.
