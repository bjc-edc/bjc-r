The solution to problem 1 needs only one if statement, because all integers are either even or odd. If the boolean expression (n mod 2 = 0) evaluates to false, the number is not even, so it must be odd.

The way problem 4 was approached above used several if statements because it tested for several conditions.

  1. n mod m reports the remainder when n is divided by m. That remainder is 0 only when n is a multiple of m. So, reports 0 only when the sprite's direction is a multiple of 60°. So, the predicate is true only when direction is 0, 60, 120, 180, 240, or 300.

    When the sprite's direction is not a multiple of 60, reports a number other than 0. For example, if the sprite's direction is 127°, reports 7.
