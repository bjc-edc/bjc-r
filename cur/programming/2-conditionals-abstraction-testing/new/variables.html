Variables look like reporters, and both do report values, but they are different.

  • Variables always give the same value until the value is changed by the user or programmer.
  • Reporters may report different output but always by same process until the process is changed by the programmer.
Reporters always have the same behavior as was coded in at the beginning. Reporter functions it can't be changed by the user but only by a programmer changing the code. Variables, the other hand, can change while the program is running.
