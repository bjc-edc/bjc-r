Mandala Design

In this project you are going to create a geometric pattern called the “Mandala” and explore its variations.
  1. Discuss with your partner how do you think this design was created.

Notice that the drawing of this design can be broken into the following steps:

  1. Trace a regular polygon (dodecagon, a 12-sided polygon is pictured in this case)
  2. Store the locations of the vertices of the polygon
  3. Draw segments from each vertex of the polygon to all the other vertices of the polygon.
Note that color to diagonal segments is added for clarity to show each stage of the drawing.
     
  1. Start by creating some custom blocks that will help keep track of important information. The design is based on a set of evenly spaced points (the vertices of the polygon), so there needs to be a way to store the locations of these points. First create a reporter block that will store in a list the input x and y coordinates of a point.
