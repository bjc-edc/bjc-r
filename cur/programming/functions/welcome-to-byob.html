Make Your Own Block: A Tutorial

We are going to teach the computer how to draw a square using a block named draw square. Please follow the steps below:
  1. Click on make a block at the bottom of the variables tab (or right-click on the scripting area and choose "make a block")
  2. This will open up the make a block dialog box. Now, you get to choose which tab the block should go into. Our block is going to draw a square, so let us choose Motion.


  3. When we selected Motion, the block became blue. We now have the option of making blocks of different shapes. Right now, however, we are just going to make a (regular) command block.



  4. When we click OK, we should see the block editor below. (For now, do not worry about the atomic check box.)



  5. Use the blocks from the regular menus to create a script that draws a square, as shown below.



  6. When you click OK, you should be able to use this block as if it were a regular block. Since you created the block as a Motion block, it will end up at the bottom of the Motion tab.



    7. Congratulations! You have just created your own block! Go on and take a bow. :-)

Improving the draw square block

You have created a block that draws a square, but it only draws a square where each side is of length 100 steps. It would be great if we could specify how long we wanted each side to be. We will edit the block to accept an argument (or input), which tells it the length of the square it has to draw.
  1. We are going to go back and edit the block. Right-click on the new block and select edit to go back to the block editor.



  2. In the Block Editor, notice that when you move the mouse over the top row of the new block, some plus signs (+) show up. When you click on these plus signs, you can add more text or arguments. When you click on the text between the plus signs, you can delete or modify that text. Click on the plus sign at the far right as shown below:



  3. When you click on the plus sign on the far right, you should get the following dialog box. With this dialog box, we can select if we want to add input (orange) or more text (blue). We want to add the input size, so we type size, select Input Name and click OK.


  4. Now, we have a variable inside our block definition.



  5. Drag the variable size down into the move block. Whenever we need a new copy of a variable, we just grab the copy from that variable in the top row.



  6. When we click OK, we wll see that our draw square block now takes an argument. We can put different numbers in the blank and draw squares of different sizes!
