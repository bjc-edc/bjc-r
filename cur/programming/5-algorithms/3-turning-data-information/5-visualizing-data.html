Grouping Data

Another thing that's often done in data science is grouping data. For example, here is the cars data grouped by vehicle make (column 12):

image needs alt-title text --MF, 6/26/19

  • Column A shows all of the vehicle makes (field 12 of each record).
  • Column B shows the total number of vehicles of each make.
  • Column C contains a list of all the data from cars for the vehicles of that Make (such as all the data for the Acuras or all the data for the Nissans). If you double-click one of the lists in column C, another table will open showing the data for all cars of that make.

Group is a higher order function like keep, map, combine, and find first because it takes a function (here, field) as input.

Plotting Data

image needs alt/title text. --MF, 6/27/19
The histogram function works like the group function, except that it allows you to specify the upper and lower boundaries of the data you are grouping by and the size of the groupings, and it sorts the data by grouping. For example, here is the cars data grouped by MPG (column 1):
  • Now, Column A shows MPG (the first field of each record) grouped into intervals of 5 and sorted.
  • As before, Column B shows the total number of vehicles within each MPG range (0–5, 5–10, 10–15, etc.)
  • As before, Column C contains a list of all the data from cars within that MPG range (such as all the data for the 268 cars that get between 20 and 25 MPG in the city).

Histogram is a higher order function like keep, map, combine, find first, and group because it takes functions (here, both field and +5) as input.
