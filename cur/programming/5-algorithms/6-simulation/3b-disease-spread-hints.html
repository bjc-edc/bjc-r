Disease Spread Hints

You can use the student file linked here to get started. Here are some hints for one possible way to write the code:
  1. Initialize the variables that will keep track of time and the number of infected. Ask the user for the, population size, the percentage of the population initially infected and the speed at which the agents will be moving.
  2. Use create a clone of myself block in Snap! to generate the population of size input by the user.
  3. Keep track of time until the entire population is infected.
  4. Initialize the clones by giving them sick (red) or healthy (blue) costume. Assign sick costumes to clones until the user specified percentage of the population initially infected is reached. Give random direction and location to clones.
  5. Once the timing starts, clones will move until they hit an edge where they bounce. If a healthy agent collides with a sick one, it will get sick.
  6. Detect collision with a sick agent.

    The collision detection code above may look needlessly complicated, as the block already exists in the Sensing menu. We create the collideWithSick? block as such because Snap! can be very slow in sensing touching by color but is very fast sensing touching by sprite.
  7. Test your code to make sure that everything works as intended.
