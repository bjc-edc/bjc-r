Disease Spread Hints

You can use the student file linked here to get started. Here are some hints for one possible way to write the code:
  1. Initialize the variables that will keep track of time, number of infected, population total and speed at which the agents will be moving. Ask the user for the percentage of the population initially infected.
  2. Use create a clone of myself block in Snap! to generate the population.
  3. Keep track of time until the entire population is infected.
  4. Initialize the clones by giving them sick (red) or healthy (blue) costume. Assign sick costumes to clones until the percentage of the population initially infected -which was input by the user- is reached. Give random location and direction to clones.
  5. Clones will move until they hit an edge where they bounce.
  6. Detect collisions between agents and if a healthy agent is colliding with a sick agent, switch the healthy to sick. Make sure that you capture the correct red color when using the touching -color- ? block.
  7. Test your code to make sure that everything works as intended.
