A Distinct List Algorithm

EK: 4.1.2A, 4.1.2B

Suppose you have a list of numbers and want to know if the numbers in the list are distinct, with no repetitions. This kind of question pops up frequently: for example, a Web search engine would want to be sure all of the search results are different from each other. Here is one algorithm to solve the problem:

Algorithms can be expressed in natural language or in pseudocode, text that describes the steps a program might carry out. These languages for human understanding may help write the algorithm in a programming language.
  • Step 1. Take the first number of the list. Compare it with the numbers later in the list (the second number, the third, etc.). If you see the starting number again, report that the numbers are not distinct.
  • Step 2. If you complete Step 1 without stopping, take the second number of the list. Compare it with the numbers later in the list (the third number, the fourth, etc.). If you see it again, report that the numbers are not distinct.
  • Step 3. Repeat Step 2 for each number in the list. Take that number and compare it with the numbers later in the list.
  • Step 4. If you complete Step 3 without finding any duplicates, report that the numbers are distinct.
