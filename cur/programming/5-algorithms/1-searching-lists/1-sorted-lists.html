Guess My Number

In this lab, you will investigate various algorithms for finding things in a list.

On this page, you will improve your number guessing game from Unit 2 to make the computer guess your number using the best strategy.

    You built a number guessing game in Unit 2, Lab 2: Developing a Number Guessing Game with Script Variables.
  1. Click the picture to load a version of the number guessing game in which you guess a number from 1 to 60.
  2. Play a few times. Keep track of how many guesses it takes you to identify the number each time.
  3. Develop a strategy that will identify the secret number in no more than 6 guesses.
  4. Write down your strategy clearly enough for others to use it to play the game.
  5. Then compare your algorithm with another pair of students.
  6. Now make the computer guess your number using this strategy.
    There are three sprites with costumes ("too small, "just right," and "too big") that are hidden in your project file. Use the show block in the "Looks" menu to see them.
  7. Save your work.
