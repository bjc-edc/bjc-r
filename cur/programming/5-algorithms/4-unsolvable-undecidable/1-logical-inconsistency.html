A proof by contradiction is a two-step proof that something is false that is done by:

  1. assuming that it's true
  2. showing how that's impossible (that it creates a contradiction)

Problem 6 asked you to devise a test question to determine whether the person you've met is a Truthteller or a False-Teller. Some statements, as you will see on the next page, could be true or false but we can prove that there is no way to test it.

There are four kinds of true/false statements:
  • Determinate, true: "Eve is a Truth-teller."
  • Determinate, false: "Eve is a False-teller."
  • Undecidable: "Adam is a Truth-teller."
  • Paradoxical: "This statement is false."
