Graphing Functions

    1. building the function-grapher using the same PlotPoint
    2. putting x in your hands
    3. rescaling

The same PlotPoint and set window scale can be used for graphing functions, but you'll need to build a different grapher that doesn't expect a list of points but instead takes a function as input and runs that function with varying values of x to generate the points.

You already have a tool that will translate the mouse position on the stage into appropriate x values based on the scale of the screen. So the position of your mouse can represent x.

PlotPoint can use that as x, and so can your function. And your function's output is PlotPoint's y. So all you need is to be able to "call" that function with the right input, and use its value in PlotPoint.

You are used to placing reporters (including functions) in input slots, but what you get is the value of the function at that moment, not the function itself, that you can evaluate at many different moments. To give a function as input, you must define it to expect a function as input.

    Leave one slot in your function blank to indicate where the value of "x" will go. The block squared shown here is not a Snap! primitive; if you want something like that, you will need to build it yourself.
