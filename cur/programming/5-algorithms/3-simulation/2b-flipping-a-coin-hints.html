Flipping a Coin Hints

Here are some hints for one possible way to write the code:
  1. Create variables Heads, Tails, Total to keep track of the number of Heads, Tails and “Total” that have been tossed, and initialize these variables.
  2. Create a custom block TossCoin that simulates a coin being tossed by rapidly switching between Heads and Tails costumes a few times.
  3. Ask the user how many times they wish to toss the coin, and check that the value entered by the user is a number.
  4. If the value entered is a number, repeat the simulation of the coin toss for the number of times entered by the user.
  5. After each toss, randomly chose between costumes Heads or Tails.
  6. Increment the associated variables Heads, Tails, and update the value of Total.
  7. Play a sound that signals the end of a flip.
  8. Add a brief pause between each toss.

If you need further hints, the following are some blocks already built for you, but put in scrambled order. Your job at this point is to put the pieces of the puzzle together to build a functional code in Snap!
