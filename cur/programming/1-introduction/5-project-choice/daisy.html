Daisy Design

In this project you are going to use Snap! to create a geometric pattern called the “Daisy Design” and explore its variations.

  1. Discuss how do you think this design was created. What are the fundamental building blocks? What kind of geometric transformations are employed?
  2. The entire design is made up of circles. So the first thing to do is to write code that will create a circle. Create a custom block to do this, via the “Variables” menu “Make A Block” option.

    The sprite moves 1 step, turns 1 degree and repeats this 360 times to make a full cycle and draw a circle. The warp block (from the Control blocks Menu) is there to make sure that the entire enclosed code is executed at once without a screen refresh. In the absence of the warp block the circle will draw slowly step by step instead of all at once.
  3. Now that you can draw a circle, generate the “Daisy Design” by rotating a bit at the end of each circle drawn. In the design given there are 24 circles, which means we turn by 360/24= 15 degrees each time.
