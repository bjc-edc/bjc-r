Robot in A Simple Maze

In this project you are given a Snap! file linked here with some custom made Motion blocks that allow a Robot to navigate in a simple maze. Five mazes have been created for you. Your job is to write code for the Robot that will allow it to escape each maze. The code should be as simple, concise and elegant as possible.

    For each of the following mazes created for you (see the Board sprite), write the shortest and most elegant code possible for the relevant escapeMaze block that will allow the Robot to escape that particular maze. Compare solutions as a class.




  1. Create your own maze and your own Robot escape code for that maze. Study and edit the drawYourMaze code in the Board sprite. If you change some of the 0’s to 1’s you will notice that you will be creating a path for the Robot to travel in the maze. It is important to understand that the boardMatrix is a list of lists and each list of 0’s and 1’s will correspond to a row of gray and white squares in the maze. Play around with the boardMatrix until you get a maze you like, then write the escapeYourMaze code for the Robot.

  2. Can you create intelligent code that will allow the Robot to escape any given simple maze? By "simple" we can understand it to mean that there is a single path from start to finish and that there are no paths that lead to nowhere. This challenge is in the realm of the study of Artificial Intelligence (AI).
