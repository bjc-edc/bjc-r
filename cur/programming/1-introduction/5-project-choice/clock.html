Build Your Own Clock

In this project you are going to use Snap! to model and build a simple clock that displays the actual time. You will write code that will make the second hand, minute hand and the hour hand of the clock to rotate properly and keep track of time.

  1. Study the Snap! file linked here to get a feel for the type of simple clock we are building in this project.
  2. Discuss and discover the mathematics behind the mechanism of a clock. How many degrees are there between each tick mark on the clock? How many degrees must the second hand of the clock rotate every second as it moves between the tick marks? How should the minute hand rotate every minute? How about the hour hand, every hour?
  3. Study the Snap! student file linked here. Notice that three sprites with costumes for the second, minute and hour hands of the clock have already been created for you. There is also a sound file for “clock ticking” included with the second hand sprite.

  4. Try to model and build the clock on your own writing code for each sprite. Look at the hints on this page only if you are really stuck.
  1. Check the accuracy of the clock you have built. For example, check it against the digital watch of a mobile phone.
  2. Does the clock you built show the hour hand moving continuously or jumping all at once at the end of every hour? The hour hand shown below moves continuously and is pointing between 12 and 1 when the time is 12:26:36. Can you adjust your code for the hour hand to implement this more realistic effect?

  3. Design and build your own clock from scratch. Find your own style and build a designer’s clock/watch putting different costumes together for the shape and the hands of the clock. Note that you may want to use an image editing software such as Adobe Photoshop to edit and properly orient the images you will use. Also make sure that the hands of the clock are anchored properly in Snap! by setting the rotation center at the appropriate location as shown below.
  4. Create an alarm clock. It should take some input from the user and then start beeping/ringing when the time is up.
  5. Build a “Grandfather Clock” that utilizes a swinging pendulum in its mechanism. Or a “Cuckoo Clock” that has an animated bird that shows up at regular intervals.
