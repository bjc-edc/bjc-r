Greeting the Player

Your Click Alonzo script runs when Alonzo is clicked. On this page, you'll make a second script that welcomes the player when the green flag () is clicked.

Blocks that look like this can be used to detect an event such as clicking on the sprite, pressing a key, or clicking the green flag button. They tell the script to start when that event occurs. Because these blocks can sit only on top of a script (as a way to start the script), Snap! programmers call them "hat blocks."

Clicking the green flag () starts all scripts that begin with the when green flag is clicked hat block. You can stop all running scripts by clicking the red stop sign () at the top right of the Snap! window.
