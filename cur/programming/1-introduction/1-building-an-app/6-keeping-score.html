Make It a Game

On this page, you'll make your app more like a game. Each time the player clicks on Alonzo, he will become a little lighter. The goal of the game is to click on Alonzo until he disappears.

We could use a video here that shows the game being played. Use alonzo.xml in the prog/1-introduction folder. Well, that one has to be stripped of the if-else for right here, but that could be put back and it would make a nice enhancement for the TIF. I'd like to re-learn how to do this.

  1. To make Alonzo keep moving around, you need a block that says "do this forever." And there is a block that does it:

    Modify your Alonzo script so that it keeps going.
    Script pic needs to be changed: input to y must vary only -130 to 130. Current inputs run off the screen.
    The sequence of blocks inside the forever block will repeat until you click the red stop sign or stop that script in some other way.
  2. Alonzo moves too fast! Use the block to slow him down. You can increase or reduce the wait time if you like.
  3. The command controls Alonzo's brightness. Experiment with different inputs to see exactly what it does.
  4. Then, modify your script so that every time Alonzo is clicked, he gets a little lighter.
  5. Play your game a few times. Adjust your code to get a level of difficulty that you like.
    6. You can use the ghost effect and the wait time and to make the game easier or harder.
