Programming Your App

On this page, you and a partner will program a simple game together on one computer. The player will try to click on Alonzo while Alonzo moves randomly around the stage.
  1. Make sure you have Snap! open (http://snap.berkeley.edu/run) and that you are logged in.
  2. Create two "Alonzo" costumes for your sprite...
    I removed the video because it does not match the current version of Snap!. --MF 2/13/17
    1. Select "Costumes..." from the File () menu, choose "Alonzo," and click "Import" and then "Cancel."
    2. Right-click (on a Mac, control-click) the Alonzo costume in the center Costumes area (not either Alonzo image on the right) and select "duplicate."
    3. Right-click (or control-click) the new costume called "Alonzo(2)", and select "edit."
    4. Click the "flip ⟷" button, and then click "OK."
  3. Program the sprite to change costumes when it is clicked...
    1. Click the "Scripts" tab above the center Costumes area.
    2. Drag blocks into the center Scripts area to build this script. For more support, read the yellow box below.

      You can find blocks in palettes with their matching color: purple for Looks, yellow for Control, etc.

      Drag one block underneath another to snap them together. The thin white bar shows where the blocks will snap together if you release the mouseclick.

  4. Check your program. Click on Alonzo on the white stage. If your script works, Alonzo should face the other way.

Saving Your Work

It is good practice to save regularly so that you don't lose the work that you've done.

  1. If you are not logged in, log in now. Then...
    "U1L1-ClickAlonzo"
    Choose "Save" from the File menu (), and give your project a name that will help you remember what the project is. For this project, use U1L1-ClickAlonzo.
    Only one partner needs to save now. You'll share the project with both partners at the end of this lab.
