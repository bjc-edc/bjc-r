On this page, you will learn that sprites can have multiple costumes and you will learn how to change them.
  1. Make sure one of you is logged in to Snap!.
  3. Program the sprite to change costumes when it is clicked. Drag blocks into the center Scripts area to build the script shown here. (For more support, read the yellow box below.)

    You can find blocks in palettes with their matching color: purple for Looks, yellow for Control, blue for Motion, etc.

    Drag one block underneath another to snap them together. The thin white bar shows where the blocks will snap together if you release the mouse button.
