Pair Programming Swap

In this course, you will use a technique that professional programmers call pair programming—writing computer programs in teams of two. At any moment, one partner is controlling the computer, and the other partner helps plan and helps catch errors.

Switch roles often, aiming for equal time in each role.

  1. If you haven't swapped yet, swap now.
