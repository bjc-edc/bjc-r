Sharing Your Game

The game isn't nearly finished, but it's finished enough for you to take a break from programming and learn how to share it with friends.

  1. Get a web address to share your app...
    I think the following should be moved to a "narrower" box. Agreed? --MF, 8/12/19
    1. Make sure your project is saved.
    2. Choose "Open..." from the Snap! File menu.
    3. Select the file from the project list (in this lab, "U1L1-ClickAlonzo").
    4. Click "Share" then click "Yes" to share. Then click "Cancel" to close that menu. (It doesn't matter which.)

      5. To locate the file you are searching for, you can type "U1" in the search bar to see only a list of your Unit 1 files.

    5. When you share the project, the URL (web address) at the top of the browser window changes. Copy the entire new URL from the top of your browser window. Email that link to a friend to share the app.
      6. Click for images that show where the URL is and how it changes.

      If you change something in the project after sharing it, the changes are reflected in what other people see. You don't have to re-share it.

  2. If you are working with a partner: Open the cloud menu () and choose "Logout." Don't close the browser window. Let your partner log in. Save the project in the second partner's account so that you both have a copy.
    The partner has a copy—not a shared file. So, if you change the project in one person's account, it will not change in the other person's account; you will need to save it to their account again for them to have the change.
