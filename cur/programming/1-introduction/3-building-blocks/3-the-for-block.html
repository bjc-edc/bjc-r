The For Block

In the current version of snap, the for block isn't installed automatically. Click on the File menu at the top of the snap window and select "Import Tools." You'll need to import tools once into each new snap project where you need for.

You have already used three ways to repeat a set of actions. In this lab, you will learn a fourth useful way.

  • The block tells the computer to repeat the same action forever.
  • The block lets you repeat an action a specific number of times.
  • The block lets you repeat an action until some special condition occurs.

The block gives you a variable that says which repetition you are on. You can use in the script you are repeating.
