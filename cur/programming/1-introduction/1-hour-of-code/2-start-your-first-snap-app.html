Start Your First Snap! App

Program the yellow Alonzo sprite to turn around when it is clicked.

  1. Use the URL http://snap.berkeley.edu/run to launch Snap! in a new window (or tab).
    Bookmark or write down the snap link. You will need it every day.
  2. Create two costumes for the sprite (one looking in each direction) by following these instructions or just watching the animation below.
    1. To make the first costume, select "Costumes..." from the File menu, , choose "alonzo.gif". The yellow Alonzo sprite will appear in the staging area.
    2. To make the second costume, right-click (or control-click on a Mac) the Alonzo costume in the staging area, select "duplicate," right-click (or control-click) the second costume, select "edit" to open the "Paint Editor," click the "flip ⟷" button, and then click "OK."
    This animation shows how to set up the costumes for the sprite.

  3. Drag one block underneath another to snap them together. The white bar shows where the block will snap.

    The blocks' colors match their palettes: purple for looks, yellow for Control, etc.
    Click the "Scripts" tab above the staging area, and build this script. When you have built the script, click on it to see what it does.

    The animation below shows where to find the blocks and how to snap them together.
    If your script works when you click on Alonzo, it should face the other way.
