Step 2: Make Alonzo move to a random spot when clicked

Now, add blocks so that every time Alonzo is clicked it moves to a random position (between -190 and 190 in the x direction and between -130 and 130 in the y direction).

Oval-shaped blocks like can report their values as inputs to other blocks. To use an oval-shaped block that way, drag it into the blank input slot in another block.

One good way to find out what a block does is to click on it and try it out.

Also remember that you can right-click on it (control-click on a Mac) and choose "help..."

Step 3: Welcome the visitor by having Alonzo say a message

Have Alonzo move to (190, -130) and say a welcome message whenever the green flag is clicked (the green flag is at the top right, above Alonzo and below the "Next" button).

The message should also tell the user to click on Alonzo and then disappear when the user clicks.

Using the Say block with a blank input will get Alonzo to say nothing.

Clicking the green flag is one way to start a Snap! program. You can stop the program by clicking the red stop sign.

In Snap!, two scripts can run simultaneously. You now have one script that runs when you click on Alonzo, and one that runs when you click on the green flag.
