Finish Your First Snap! App

One good way to find out what a block does is to click on it and try it out.

Oval-shaped blocks like can report their values as inputs to other blocks. To use an oval-shaped block that way, drag it into the blank input slot in another block.

    If you want to know more about what a block does, you can right-click it (or control-click on a Mac) and choose "help..."
  1. Add blocks so that every time Alonzo is clicked it not only turns, but also moves to a random position (between -190 and 190 in the x direction and between -130 and 130 in the y direction). Use the and blocks togther at the end of your script to do this.
    2. The Snap! stage (the white space in the upper right where all of the stuff happens) is just like a coordinate plane in math class. The x axis goes from -190 to 190, and the y axis goes from -130 to 130.
    Clicking the green flag is one way to start a Snap! program. You can stop the program by clicking the red stop sign.
  2. Have Alonzo move to (190, -130) and say a welcome message whenever the green flag at the top right, , is clicked.

    3. The message should also tell the user to click on Alonzo and then disappear when the user clicks.

    In Snap!, two scripts can run simultaneously. You now have one script that runs when you click on Alonzo, and one that runs when you click on the green flag.
Scripts run their blocks from top to bottom. Clicking on any script runs the script, but you want this script to run when you click on Alonzo.

Using the Say block with a blank input will get Alonzo to say nothing.
