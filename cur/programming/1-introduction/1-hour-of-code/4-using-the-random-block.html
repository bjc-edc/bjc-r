Using the Random Block

Next, you'll add blocks so that every time Alonzo is clicked, it moves to a different random position using the random block.

The random block generates a random number within a range (defined by two integers). For example, each time the block is run, it reports one number ranging from -10 through 20 (with about equal likelihood).

    One good way to find out what a block does is to click on it and try it out.
  1. Find the block in the green Operators palette, and click it to try it out.
  2. Discuss the values that the random block reports. What do you notice?
