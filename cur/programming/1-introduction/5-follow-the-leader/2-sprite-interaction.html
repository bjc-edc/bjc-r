AAP-2.K.3

The language used on the AP Exam doesn't allow spaces for names of inputs (such as number of fish) or programmer-defined procedures (such as mouse y, which isn't built into their language).

The reason for the 
()
after 
MouseY
is that it's a procedure call even though it doesn't take any inputs.

The script

would be written as

REPEAT UNTIL (MouseY () < 0)
{
    DISPLAY (numFish)
}
or

Remember, you don't need to learn to write the made-up language used on the AP exam. You just have to be able to read and answer questions about it.
