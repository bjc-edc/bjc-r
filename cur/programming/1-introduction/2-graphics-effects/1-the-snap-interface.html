The snap Window   (optional)

Naming the sections of the snap window makes it easy to refer to them.

Palette

The palette (left side of snap window) contains the blocks that you use to tell the computer what to do. The blocks are organized into eight categories according to the kind of thing they do: Motion, Looks, Sound, Pen, Control, Sensing, Operators, and Variables.

Scripting Area

You drag blocks to the Scripting Area to create scripts to control the computer.

Stage

The Stage is where sprites—little on-screen robots—act out the code you write. The sprite () in the picture above is saying "Hello World!" because that is what its script told it to do when the green flag was clicked. You can change the background design of your stage depending on what you want your program to do.

Sprite Corral

The Sprite Corral houses all of the sprites in the project, as well as an icon representing the stage. The project shown here has only one sprite.

Tool Bar

  • the File menu (for saving and loading your projects)
