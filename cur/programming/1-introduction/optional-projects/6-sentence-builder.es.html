Modelando Lenguaje

Antigua Página Modelado de Estructuras de Oraciones

En este proyecto, usará reporteros y listas para crear un programa capaz de crear un lenguaje más complicado que el proyecto Gossip.

    • sustantivos (cosas)
    • verbos (comportamiento)
    • adjetivos (descriptores de cosas)
    • adverbios (descriptores de acción)
    • determinantes (identificadores—le dice qué cosa)
    • preposiciones (indicadores de relación)

  1. Contiene seis reporteros, cada uno de los cuales informa una palabra aleatoria de una lista de palabras en esa categoría.

  2. Pruebe cada bloque para ver lo que hace. Luego cambie estas listas como quiera para incluir sus propios sustantivos, verbos, etc.
  3. Cree un reportero llamado oración simple que reporte una oración simple usando el determinante aleatorio, sustantivo aleatorio y verbo aleatorio bloques
  4. Se pueden construir oraciones más complicadas a partir de varias frases. Crea estos reporteros de frases:
    1. debe unir palabras de dos categorías para formar frases como "el sándwich" o "una jirafa", o de tres categorías para crear frases como "el niño pequeño", "una jirafa verde" o "nuestro viejo sándwich".
    2. debe combinar una preposición con un sintagma nominal para crear frases como "cerca del niño pequeño" o "sobre una pizza sabia"."
      3. Usar frase nominal para definir frase preposicional es un buen ejemplo de abstracción. Si luego cambia frase nominal para incluir nombres de personas, frase preposicional seguirá funcionando.
    3. debe reportar solo un verbo como "saltos" o une un verbo y un adverbio para crear combinaciones como "salta tristemente", "duerme la siesta rápidamente" o "piensa con sueño".
    5. Estas frases serán gramaticalmente correctas, pero no necesariamente tendrán sentido. Por ejemplo, sintagma nominal podría generar "una pizza cansada".
  5. Luego, haz una oración complicada de reportero que combine un sintagma nominal, un sintagma verbal y un sintagma preposicional.
  1. Ocasionalmente, incluya nombres de personas en lugar de una frase nominal. Entonces, en lugar de algo como "mi tonto elefante", el programa a veces podría usar "Jamie" u otros nombres.
  2. Las frases nominales no siempre tienen que tener un adjetivo. Pueden tener ninguno, o dos, o más. Añade algo de variedad.
