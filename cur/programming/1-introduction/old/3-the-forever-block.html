The Forever Block

As the name "forever" implies, the forever block will run the blocks inside it again and again and again and again... forever. You can stop one script by clicking on it again. You can stop all scripts by clicking on the stop sign at the top right corner of the snap window. This is equivalent to executing the in the "Control" palette.
