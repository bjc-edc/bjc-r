Operators

Math operations such as and are reporters. Operations nested inside each other are just like parentheses in expressions in algebra:

= 5 + (4 · 3) = 5 + 12

The order of operations depends on how the blocks are embeded. Here, snap will first evaluate the innermost block, 4 · 3, which equals 12, and then evaluate 5 + 12. If you want to change the order of operations for an expression, rebuild the expression with the multiplication block on the outside.

The Mod Block

The block reports the remainder when the first positive integer input is divided by the second. For example, because when 17 is divided by 5, the remainder is 2.

In mathematics, we write as 17 (mod 5) ≡ 2 and we write as 15 (mod 5) ≡ 0.

When one number divides another evenly, the remainder is 0: . So, the predicate reports if n is divisible by 2 (so if n is even) and if n is odd (not divisible with 2 and so the remainder is 1).
