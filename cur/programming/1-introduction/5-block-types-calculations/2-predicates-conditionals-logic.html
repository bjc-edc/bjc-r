Conditionals 

Conditionals are found in the yellow Control palette with repeat, forever, and for.
  • Conditional blocks are a special kind of command. Conditionals perform the code inside them if (and only if) the condition input (a Boolean value) is
    • true for the first command slot of an if.
    • false for the second (else) command slot of an if or the command slot of repeat until.
