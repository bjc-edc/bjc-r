Getting to Know the Snap! Interface

Naming the sections of the Snap! window makes it easy to refer to them. This description is brief. You can always look up more detail, if you like, in the Snap! manual.

Palette

The palette contains the blocks (individual instructions) that you use to tell the computer what you want it to do. The blocks are organized into eight categories according to the kind of thing they do.

Scripting Area

You drag blocks to the Scripting Area to create scripts to control the computer. The script shown here has two blocks: when GreenFlag is clicked and say Hello!

Stage

The Stage is where sprites—little on-screen robots—act out the code you write. This sprite is saying "hello" because that is what its script told it to do when the green flag was clicked.

Sprite Corral

The Sprite Corral houses all of the sprites that you (or someone) created. You start with one sprite and can create more if you like. When a sprite in the Corral is selected, the Scripting Area shows what instructions that sprite understands. The picture here shows only one sprite in the corral.

Tool Bar

The Toolbar across the top of the screen has buttons for the File menu for saving and loading your projects, the Cloud menu for accessing your Snap! account, the Settings menu with advanced options, stage resizing buttons , and project control buttons (green flag , pause , and stop ). You can click on the buttons to see more about what they do.
You can also click on the Snap! logo for more information.
