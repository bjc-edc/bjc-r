What is Snap! ?

Snap! is a language for writing computer programs. Programs are sets of instructions for the computer to follow, telling it what to do.
When the first programming languages were invented, video screens were rare, and people communicated with computers mostly through a keyboard. Because of that history, most programming languages still use a keyboard. Snap! is newer: it's a visual language. Instead of writing a program using only the keyboard, you will build the program by dragging pictures of blocks (instructions) and clicking them together.

Introduction to Pair Programming

In this course, you will use a technique that professional programmers call pair programming—writing computer programs in teams of two people. One partner, the "driver," writes the program that controls the computer; the other partner, the "navigator," helps plan and helps catch errors.
Switch roles often, trying for equal time in each role. That way each partner gets to understand all parts of the program, and the person in each role remains focused. As you progress through labs, we will suggest places for you to switch.
