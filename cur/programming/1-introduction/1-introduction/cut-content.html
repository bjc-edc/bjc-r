Saving

That didn't save your work. It just created a space for your work. Now, use the File button in the tool bar to save your work so far. Give it a name that will help you remember what the project is.

Any time you want that work back, you will use the same button and select "Open..." to see a list of your files.

What is snap ?

snap is a language for writing computer programs. Programs are sets of instructions for the computer to follow, telling it what to do.

When the first programming languages were invented, video screens were rare, and people communicated with computers mostly through a keyboard. Because of that history, most programming languages are still written with text using a keyboard.

snap is newer: it's a visual language. Instead of writing a program using only the keyboard, you will build the program by dragging pictures of blocks (instructions) and clicking them together.

Introduction to Pair Programming

In this course, you will use a technique that professional programmers call pair programming—writing computer programs in teams of two people. One partner, the "driver," writes the program that controls the computer; the other partner, the "navigator," helps plan and helps catch errors.

Switch roles often, trying for equal time in each role. That way each partner gets to understand all parts of the program, and the person in each role remains focused. As you progress through labs in the first few units, we will suggest places for you to switch.
