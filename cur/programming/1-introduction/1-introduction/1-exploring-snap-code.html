Exploring snap Code

Programs are sets of instructions that tell a computer what to do.

You have already programmed an app in the snap programming language by dragging blocks (individual instructions) and snapping them together into scripts.

Now you will experiment with some more blocks to see what you can do.

Bookmark or write the snap link. You will need it every day.
