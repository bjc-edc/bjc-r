Pair Programming Swap

In this course, you will use a technique that professional programmers call pair programming—writing computer programs in teams of two people. One partner writes the program; the other partner helps plan and helps catch errors.

Switch roles often, aiming for equal time in each role. If you haven't swapped yet, swap now.
