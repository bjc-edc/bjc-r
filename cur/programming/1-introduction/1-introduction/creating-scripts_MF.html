In the scripting area, you can snap command blocks together (hence the name snap) to build up a set of instructions, a script. Drag one block right underneath another to snap them together. Blocks in a script run in a specific order, from the top of the script to the bottom. Generally, snap waits until one block has finished its job before continuing on to the block below it. (One common exception is blocks that play sounds: a block's job can be to start the sound, which means the block below it will execute while the sound is still playing.)

Reporter blocks can report many kinds of values. The plus block, for example, reports a number.

Some reporter blocks, like greater than report only two special values: when a condition is met and and when it is not met.

Reporters that report only true and false are called "predicates." In snap, these functions have the hexagonal shape: .

Other predicates you will often find useful equals, not, and, and or include

Predicate functions are like questions that can be answered with true or false. These two special values are called Boolean values, named after the logician George Boole.
