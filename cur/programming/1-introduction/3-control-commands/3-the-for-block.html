The For Block

Programs often require repeating a set of actions. You have already used three ways and will learn a fourth in this lab session.

  • The block is appropriate when you want to repeat the same thing forever.
  • The block is appropriate when you want to repeat the same thing a specific number of times.
  • The block is appropriate when you want to repeat the same thing until some special condition occurs.
  • The block is appropriate when you want to repeat almost the same thing with a change in each step.
