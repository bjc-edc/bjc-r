Making a Custom Block

Up until now, you used blocks that already existed. You will now learn how to build your own blocks. As an example, you will start by making a block that draws a square and you will name this new block draw square. Here's how:
  1. At the bottom of the "Variables" palette, click on "Make a block." (As a short-cut, you can right-click or control-click on the scripting area background and choose "Make a block"). This opens up the "Make a block" dialog box.
  2. Every block needs to have a color (a category), a title, a shape (you've seen Commands and Reporters and will later see the third shape, Predicates), and a script that defines its behavior. You will choose these next.
    1. Tab: Choose the tab you want your new block to go into. You can put it anywhere you like, putting it in "Motion" with the block that draws a line would make sense and be easy to remember. (If you don't choose a color, your block will be a gray "Other" block and will be sorted into the "Variables" tab.)
    2. Title: Then give your block a name. Type "draw square" as shown here.
    3. Type: This block is a Command, like move, not a Reporter. (You will learn about "Predicate" later.) The "Command" icon is already selected, and this is the one you want, so you don't change it.
    4. In this course, you will almost always want to keep "for all sprites" selected, so you can ignore that choice for now.
    5. Click "OK".
After you click "OK", a dialog titled "Block Editor" will appear.
The editor starts you with a hat block, a kind of block that is the start for any block you create. It contains the title of your block, draw square, in this case. Hat blocks can never be in the middle of a script, only at the top.
  1. Behavior: Build the instructions for your new block here exactly as you do in the scripting area: Drag blocks from the palette to define the script that will run when your custom block is used.

When you click "OK", you can use this block exactly the way you have used other blocks. Since you created the block as a "Motion" block, you will find it in the "Motion" tab.

Congratulations! You have just created your first custom block.
