Your First Custom Block

Up until now, you used blocks that already existed. You will now learn how to build your own blocks. As an example, you will start by making a block that draws a square and you will name this new block draw square. Here's how:

  1. At the bottom of the "Variables" palette, click on "Make a block." (As a short-cut, you can right-click or control-click on the scripting area background and choose "Make a block").

  2. This opens up the "Make a block" dialog box. You get to choose which tab you want your new block to go into. You can put it anywhere you like, but drawing a square is so much like drawing a line that it makes sense to put it in "Motion". (If you forget this step, your block will become a gray "Other" block to be found in the Variables tab.)

  3. Now you have four choices to make.
    1. Then give your block a name. Type "draw square" as shown here.

    2. This block is a Command, like move, not a Reporter. (You will learn about "Predicate" later.) The "Command" icon is already selected, and this is the one you want, so you don't change it.
    3. In this course, you will almost always want to keep "for all sprites" selected, so you can ignore that choice for now.
    4. Click "OK".

  4. After you click "OK", a dialog titled "Block Editor" will appear.

    Inside the you will see a specially shaped block with the title of the custom block that you are creating. This kind of block is called a "hat" block, and it indicates a starting place for the draw square script.

    You can treat this dialog like you do the scripting area of the main snap window. Drag blocks here from the palette to define the script that will run when your custom block is used.

  5. Use the blocks from the "Control" and "Motion" palettes to create a script that draws a square, as shown below.

  6. When you click "OK", you should be able to use this block as if it were a regular block. Since you created the block as a "Motion" block, it will end up at the bottom of the "Motion" tab.

Congratulations! You have just created your first custom block.
