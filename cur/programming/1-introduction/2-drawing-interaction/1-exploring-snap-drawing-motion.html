Drawing and Motion

A script is a sequence of instructions—an algorithm—that tells a computer what to do. A program is a set of scripts. In some programming languages, like snap, you program by dragging blocks (individual instructions) and snapping them together into scripts. In other languages, you type instructions to create scripts. The idea is the same.

For now, you're using blocks that come built in with snap. Soon, you'll invent and create your own blocks—blocks that snap does not provide.


    The blocks' colors match their palettes: blue for Motion, green for Pen, etc.
