Modify Your Pinwheel

On this page, you will modify your existing pinwheel block so that you can use it to draw a variety of shapes.

Adding Multiple Inputs

Recall how you first generalized your original pinwheel script: you added an input variable called branches that controlled the turning angle of the sprite. By adding more input variables, you can generalize other aspects of your program...

Hiding the Complexity

You can create use your general-purpose pinwheel procedure to create other, simpler special-purpose procedures that each do a particular job.

  1. Modify the code from your pinwheel block to create a more specialized asterisk block.
    1. First, make a new block called asterisk with two variables: one for the number of branches and one for the length of each branch.
    2. Then, leaving the block editor for asterisk open, edit the pinwheel block, right-click (or control-click) on the repeat block (the yellow part), and select "duplicate." (The video below shows this process.)
    3. Drag the duplicate code into the asterisk block editor and attach it to the hat block.
    4. Then close the pinwheel code editor, and edit the asterisk code so that it works as intended.
