Puedes usar el procedimiento molinillo con propósito general para crear otro, un procedimiento más simple con propósito especial que realice un trabajo concreto.

  1. Si aún no has abierto el proyecto U1L3-Molinillo de la página anterior, ahora es el momento.
  2. Usa el código del bloque molinillo para crear un bloque más especializado llamado asterisco.
    1. El primer paso es crear un nuevo bloque llamado asterisco con dos parámetros: uno es número de ramas y otro llamado largo de cada rama. Para recordar la forma de crear un nuevo bloque visitar: Unidad 1 Laboratorio 2 Página 4: Crear un nuevo bloque.
    2. Ahora introduce los valores a molinillo para que pueda dibujar correctamente un asterisco.
  3. De forma similar, ahora es el turno de escribir un bloque polígono que haga uso de molinillo con los valores apropiados para dibujar un polígono que reciba el número de lados y el largo del lado.
