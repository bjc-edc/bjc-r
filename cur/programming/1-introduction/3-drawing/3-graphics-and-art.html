Graphics and Art

On this page: You will create and use blocks that let you draw polygons, circles, stars, to explore art and graphic design.

  1. If you have not done so already, build a custom block to draw a regular polygon of a given number of sides and a given side length.
  2. You may also wish to build a custom block to draw a rectangle of given length and width.
  3. Now create, in the Snap! coding area, a palette of tools and blocks that may be handy in creating your art work. You can adjust the input values for these blocks as needed as you create your art.
  4. Watch the following video which should give you an idea how you might be able to create some overlapping polygons and then fill the regions created with color.
