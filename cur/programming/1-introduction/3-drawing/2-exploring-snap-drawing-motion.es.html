Un giro de 360° representa una vuelta completa. La mitad de eso (180°) es una media vuelta. Un cuarto de 360° es un giro de noventa grados. No es necesario hacer la división en cada oportunidad, ya que Snap! tiene una función para ello. Se puede utilizar el bloque o el bloque para una séptima parte de una vuelta, o un tercio de giro.
