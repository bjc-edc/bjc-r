Blocks with Inputs


On this page: You will create a block with an input slot, like the slots in . That lets you use that information inside the block so the one block can do many related jobs.

  1. Start by loading this file. Make 5 copies of the script and modify those copies so that they make pinwheel designs like these with 3, 4, 5, 6, and 12 branches.

The scripts are essentially all the same; only the input values change. Instead of having many separate scripts, you can make one more general script that will draw all such designs. Your pinwheel block will need an input to specify how many branches you want, just like the go to block has inputs to specify x and y values .

  1. Create a pinwheel block with an input for the number of branches. Instructions below and in the video below right.

    Making an Block with an Input

    1. Type the block name and the label for your variable in the "Make a block" dialog box, and click OK. For this project, type pinwheel, branches:
      Click here to watch a VIDEO of these steps. (If your connection blocks YouTube, watch the video here.)
      • This block is about moving your sprite, so you might choose the blue Motion palette. Or just leave it gray.
      • Using a comma and a colon are not requirement; they are here to help with clarity.
    2. Click the + (plus sign) after the variable label, type the name of the input, and click OK to add the variable input slot. For pinwheel, click the + after "branches:" and type number of branches as the name of your input variable.
    3. Drag in the blocks you need, and use the new input.
      • Here, drag one of your pinwheel scripts into the Block Editor and snap it to the pinwheel block.
      • Then, drag the new input number of branches in to replace the values that change to allow for more general use of the script.
  2. Save your work as U1L3-Pinwheel.
