The For Block

In the current version of snap, the for block isn't installed automatically. Click on the File menu at the top of the snap window and select "Import Tools." You'll need to import tools once into each new snap project where you need for and some other important tools.

You have already used three ways to repeat a set of actions.

  1. The block tells the computer to repeat the same action forever.
  2. The block lets you repeat an action a specific number of times.
  3. The block lets you repeat an action until some special condition occurs.
On this page, you will learn a fourth useful way.
