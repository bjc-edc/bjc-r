Customizing and Debugging

On this page, you will edit the word lists used by the sprites and debug any problems.

  1. Edit these three reporter blocks to change the phrases that the program uses to make sentences. Instructions are in the yellow box below. Be creative and funny, but thoughtful.

    How to Edit a Block

    • Locate the block in the scripting area or the colored palettes on the left.
    • Control-click (or right-click) on the block and choose "edit..."
    • Make changes to the block's internal code. (Instructions are already inside these blocks.)
    • Click OK when you're satisfied. (Or click "Cancel" if you don't want to make changes.)

Checking and Debugging

Whenever you program, give your programs a test drive; check to see if they do what you want.

  • Generally, you should first test parts that you've just built or changed.
  • Then, if the parts work, test the whole program.

  1. Test the blocks you just edited by clicking each of them a few times. If anything looks wrong, go back inside the block and edit it again until it works correctly.
  2. Click the to test your whole program a few times. Fix anything that still needs work.
