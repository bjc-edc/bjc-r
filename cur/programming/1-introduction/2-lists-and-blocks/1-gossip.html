Editing blocks in a working program

You can improve this program by editing who and does what to give them more (or different) language.
  1. Here's how to edit who to give it more names.
    TO EDIT ANY  BLOCK,
    • Control-click (or right-click) on that block and choose "edit..."
            In this case, control-click on who.
    • Make your changes.
            In this case, you'll find instructions inside the block.
    • Click OK when you're satisfied. (Or click "Cancel" if you don't want to make changes).
  2. Now edit does what. Be creative and funny (but thoughtful)!
