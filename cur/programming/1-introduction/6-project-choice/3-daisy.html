Daisy Design

In this project you are going to use Snap! to create a geometric pattern called the “Daisy Design” and explore its variations.

  1. Discuss how this design was created. What are the fundamental building blocks? What kind of geometric transformations are employed?
  2. The entire design is made up of circles. So the first thing to do is to write code that will create a circle. Create a custom block drawCircle to do this, via the “Variables” menu “Make A Block” option.
  3. Now that you can draw a circle, you can generate the “Daisy Design” by rotating a bit at the end of each circle drawn. Create a custom block drawDaisy to do this.

    4. If you get stuck in any of the previous steps, you can look at this page for hints.
