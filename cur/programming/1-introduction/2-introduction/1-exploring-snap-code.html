Exploring snap Code

You have already programmed an app in the snap programming language.

Programs are sets of instructions that tell a computer what to do.

In snap, you program by dragging blocks (individual instructions) and snapping them together into scripts.

For now, you will experiment with some more blocks that snap provides, to learn what they do.

Soon, you will create your own blocks of code, blocks that snap doesn't have.

  1. Build this script.

    Clicking anywhere on a script runs it.

    When you have built it, click on it to see what it does.

  2. Analyze why this script does what it does.
  3. Compare your work with another pair.

    If your script behaves differently from theirs, check both.
