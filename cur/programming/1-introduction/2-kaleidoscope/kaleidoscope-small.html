Make a Kaleidoscope

Here is a drawing program to explore. (Click the link to get it.) The main part of it is
The hat shaped blocks, which can be used only at the beginning of a script, tell when to run a specific script.

  1. You've used part of this script before and know what it does. The new part is when space key pressed. Experiment to see what that does.
  2. Now, while you are moving your mouse over the stage, use the d, u, and c keys to control what gets drawn on the stage. Can you get the sprite to write your name?
  3. Now you get to invent a script on your own to make a (kind of) kaleidoscope, like this:
    One sprite is following the mouse. Three other sprites are reflecting the movement of that sprite in slightly different ways.

    Some useful information:
    • You will need four sprites. One way to create the others is to duplicate the one you have. Right-click the sprite-button in the sprite corral, and select "duplicate" from the context menu that appears. Each duplicated sprite will have exactly the same scripts as the original, which is why we suggest duplication rather than just creating more sprites from scratch.
    • Click on any sprite in the corral to see its script. They will all be the same at first, but you want to modify three of them to change the color and create the reflections.
    • To change the color, click the color input in that sprite's block, choose whatever color you like, and then click on the block itself (to run the block and actually set the color).
    • Pay close attention to what each of the other sprites is doing in the animation above. You will need to modify the x and y inputs in each sprite's block using or .
  4. Once you figured this out, try some other formulas and/or more sprites, and share with your classmates! Or, instead of using the coordinates of the mouse, the sprites could point toward each other and chase each other by moving a short distance (and possibly also turning, as well, so that they don't instantly crash). Experiment!
  5. You've created something on your own. Save it! The File button in the tool bar will let you save your project. Give it a name that will help you remember what the project is.
