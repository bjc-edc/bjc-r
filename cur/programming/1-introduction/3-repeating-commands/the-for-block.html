The For Block

Repeating a part of your program, looping, is a very common practice in programming. There many ways to do it. You have already used two of them.
The block is appropriate if you want to loop the same behavior forever.
The block is appropriate if you want to loop the same behavior a specific number of times.
Sometimes, though, you want to do almost the same thing for a certain number of times, but with a slight variation. For instance:
  • You want the script to tell the sprite to draw 20 squares on the stage, each square larger than the previous.
  • You want the script to tell the sprite to draw 12 regular polygons, each with one more side than the previous.
In the current version of snap, the for block isn't installed automatically. You will need to import it once for each new snap project in which you use it. To do so, click on the File icon at the top of the screen and select "Import Tools".)
