  • Commands—blocks that carry out an action like drawing on the stage or having the sprite say something. Commands have the click-together groove and bump at the top and bottom.

The three oval-shaped blocks at the bottom of the Motion palette are called reporters. Unlike the jigsaw-puzzle-shaped command blocks, reporters don't carry out an action (such as moving the sprite or displaying a speech balloon). Instead, they report a value, usually for use in another block's input slot.

The and reporters tell you where the sprite is on the stage. As in algebra, x means left-to-right position, and y means bottom-to-top position.

Blocks in a script run in a specific order, from the top of the script to the bottom. Generally, snap waits until one block has finished its job before continuing on to the block below it. (One common exception is blocks that play sounds: a block's job can be to start the sound, which means the block below it will execute while the sound is still playing.)
