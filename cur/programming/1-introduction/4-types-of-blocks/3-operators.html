The Mod Block

The block reports the remainder when the first positive integer input is divided by the second. For example, because when 17 is divided by 5, the remainder is 2.

 

When one number divides another evenly, the remainder is 0 . So the predicate reports true if n is divisible by 2 (i.e., if n is even) and false if n is odd.

 

In mathematical notation, we write as 17 (mod 5) ≡ 2 and we write as 15 (mod 5) ≡ 0.
