Making a New Block

  1. Click at the end of the palette, or control-click (or right-click) in an empty spot in the scripting area and choose "make a block..."
  2. Choose the color, in this case red for Lists.
  3. Type the title for your block: .
  4. Select the shape and click OK. For this block, choose the oval reporter shape because you want it to report a value (the chosen response) to another block (the join).

    5. Because you chose the reporter shape, the block editor opens with a block already attached to the hat block. Whatever input you give to report will become the value reported by your new block.

The "value of a procedure" (technically, the value of a procedure call) is the value it returns when called. For example, if you write the procedure: 

PROCEDURE double (x)
{
    RETURN (2 * x)
}
then, "the value of 
double (5)
" means 10.
