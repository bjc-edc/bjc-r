On this page, you will use reporters and lists to create a program capable of creating more complicated language than the Gossip project.

    • nouns (things)
    • verbs (actions)
    • adjectives (thing describers)
    • adverbs (action describers)
    • determiners (identifiers—tells you which thing)
    • prepositions (relationship indicators)

  1. It contains six reporters that each report a random word from a list of words in that category.

  2. Change these lists however you like to include your own nouns, verbs, etc.
  3. Create a reporter called simple sentence that reports a simple sentence using the random determiner, random noun, and random verb blocks.
  4. Complicated sentences are built up from multiple phrases. Create three other reporters that report different kinds of phrases:
    1. should join words from three categories to create phrases like "the little boy," "a green giraffe," or "our old sandwich."
    2. should combine a preposition with a noun phrase to create phrases like "near the little boy" or "over a wise pizza."
      3. Defining prepositional phrase using the noun phrase block is a good example of abstraction. If you later change noun phrase to include people's names, prepositional phrase will still work.
    3. should join a verb and an adverb to create phrases like "jumps sadly," "naps quickly," or "thinks sleepily."
    5. These phrases will be grammatically correct, but they won't necessarily make sense. For example, noun phrase might generate "a tired pizza."
  5. Then, make a reporter complicated sentence that combines a noun phrase, a verb phrase, and a prepositional phrase.
 
  1. Occasionally, include people's names instead of a noun phrase. So, instead of something like "my silly elephant", the program could sometimes use "Alfred" or some other names.
    2. I wanted to keep this one, but I can't figure how they would handle multiple cases with the content that has been introduced so far. --MF, 9/11/18
  2. Noun phrases don't always have to have one adjective. They can have none, or two, or more. Add some variety.
