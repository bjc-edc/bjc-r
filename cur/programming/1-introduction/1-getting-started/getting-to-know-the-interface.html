Getting to Know the snap Interface

We name the sections of the snap window so that we can talk about them.

Tool Bar

The Toolbar includes the File menu for features related to saving and loading projects, the Cloud menu with options related to your snap account, the Settings menu with advanced options, window resizing buttons, and Project Control Buttons (green flag, pause, and stop). The snap Reference Manual contains more details about the Tool Bar.

Scripting Area

The Scripting Area is where you will drag blocks of code to create scripts to control the computer. The script here has two blocks: "when green flag is clicked" and "say Hello!" The snap Reference Manual contains more details about the Scripting Area.

Stage

The Stage is where the code you will write will be acted out by sprites. Sprites are computer graphics that you will use code to move around and do things on the screen. The snap Reference Manual contains more details about the Stage.

Sprite Corral

The Sprite Corral is where you create and define your sprites. It is like a library of all of the sprites you have created. The snap Reference Manual contains more details about the Spite Corral.

Palette

The area called the palette contains the blocks (individual instructions) that you will use to tell the computer what you want it to do. The blocks are organized into eight categories according to the kind of thing each block does. The snap Reference Manual contains more details about the Palette.
