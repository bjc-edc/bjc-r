You are going to work with a set of 500 real text messages, part of a collection recorded and collected by T. Almeida and J. Hidalgo (original source).

Two typical lines in our data look like this:
ham Fair enough, anything going on?
spam SMS. ac Sptv: The New Jersey Devils and the Detroit Red Wings play Ice Hockey. Correct or Incorrect? End? Reply END SPTV

The raw data is here, but we've made a starter snap project which you should open now. It has a script that you can run to import the text data (from the above link), split it line by line and then into a list with two entries, and then store it into a variable datasheet:

Be aware that this is real-world data with very casual langauge (we've already removed the messages with language that was too casual for a school assignment!)

We will call that first word—spam or ham— the ground truth classification. That is, at some point an actual human decided whether this message was spam or ham, and we'll consider this to be "the truth". In this way, we can see how good our program is at classifying a message, based only on its contents, by comparing it this value. You might actually disagree with some of the "truth" classifications that are in the file; if so, feel free to change them (and be ready to justify your belief to others!)

Out of the 500 messages, how many are ham? Write a script to find out.
