Contando tazones de helado

En esta práctica de laboratorio, explorará un problema clásico de Ciencias de la Computación: los subconjuntos.

En esta página, explorará una introducción familiar a los subconjuntos: elegir sabores de helado.

Un subconjunto es una selección de elementos de un conjunto; puede ser ninguno, todos o cualquier número intermedio.

Hay muchos problemas que tienen que ver con subconjuntos. Por ejemplo,
  • Hay 20 estudiantes en la clase y desea un comité de cuatro. ¿Cuántas combinaciones de cuatro personas hay?
  • ¿Cuántas combinaciones hay para una cerradura Simplex? (Una cerradura Simplex tiene cinco botones. Puede usar cada botón como máximo una vez, pero puede presionar más de uno al mismo tiempo. Vea la imagen a la derecha).
  • Una tienda tiene ocho sabores de helado. ¿Cuántas combinaciones de sabores de helado puedes elegir para un tazón con un número dado de bolas?

Por ahora, todas las primicias deben ser diferentes entre sí. No importa el orden de las bolas: un bol con vainilla y chocolate es lo mismo que un bol con chocolate y vainilla.
  1. ¿Cuántos tazones de dos cucharadas hay? Encuentre una forma sistemática de enumerarlos.
  2. Busque una manera de describir el problema utilizando la recursividad.
  3. "U8L3-Subsets"
  4. Es posible que desee utilizar el bloque añadir.
    Cree un bloque que, dada una lista de entrada de longitud variable de sabores disponibles, informe una lista con todos los tazones de dos cucharadas. (Dado que un tazón de dos cucharadas se representa como una lista de dos sabores, una lista de tazones es una lista de listas.)
El bloque que construiste para tazones de dos cucharadas puede ayudarte a construir un bloque para tazones de tres cucharadas.
  1. Cree un bloque que enumere todos los tazones de tres cucharadas, dada una lista de los sabores disponibles. Recuerde, el orden de las cucharadas no importa, así que piense en cómo podría evitar enumerar el mismo tazón más de una vez.
  2. Ese tazón tiene chocolate o no.
    Generalice para crear un bloque que enumere todos los tazones de n cucharadas, con una entrada adicional dando el número de cucharadas. ¡Esto definitivamente necesitará recursividad!
  1. Escriba un procedimiento que, dada una lista de sabores disponibles, reporte una lista de todos los tazones posibles sin importar el número de cucharadas.
  2. Elimine la restricción de que cada cucharada debe tener un sabor diferente: ahora se permite chocolate, chocolate y chocolate, y muchos más. (Tenga en cuenta que, técnicamente, este problema ya no se trata de subconjuntos, porque, por definición, un conjunto no puede tener el mismo elemento dos veces).
