In this lab, you will explore a classic computer science problem: subsets.

On this page, you will explore a familiar introduction subsets: choosing flavors of ice cream.

A subset is a selection of items from a set; it can be none, all, or any number in between.

There are lots of problems that are about subsets. For example,

  • There are 20 students in the class and you want a committee four. How many combinations of 4 people are there?
  • How many combinations are there on a Simplex lock? (A Simplex lock has five buttons, where you can use each one once, but you can press them at the same time. See image at right.)
  • A store has eight flavors of ice cream. How many combinations of ice cream flavors can you chose for a bowl with a given number of scoops?

For now, all scoops must be different from one another. The order of the scoops doesn't matter: a bowl with vanilla and chocolate is the same as a bowl with chocolate and vanilla.
  1. How many 2-scoop bowls are there? Find a systematic way of listing them.
  2. Look for a way to describe the problem using recursion.
  3. You may want to use the append block seen in Unit 8 Lab 2 Page 3: Partition Sort.
    Build a block that reports a list of lists, all 2-scoop bowls given a variable-length input list of available flavors.
The block you built for 2-scoop bowls can help you build a block for 3-scoop bowls.
  1. Build a block that lists all 3-scoop bowls, given a list of the available flavors. Remember, the order of the scoops doesn't matter, so think about how you might avoid listing the same bowl more than once.
  2. That bowl either has chocolate, or it doesn't...
    Generalize to build a block that lists all n-scoop bowls, with an additional input giving the number of scoops. This will definitely need recursion!
