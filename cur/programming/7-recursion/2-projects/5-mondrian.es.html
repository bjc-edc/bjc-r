  4. Luego, programe el código para el caso recursivo. El bloque Mondrian ya contiene algo de estructura. Su trabajo es averiguar qué entradas necesitará en función de cómo debería funcionar el script.

    • Ya has visto el nivel 1, un rectángulo relleno. En cada nivel, Mondrian subdivide el espacio que tiene en dos rectángulos (que se subdividirán aún más si se requieren más niveles). Por ejemplo, en el nivel 2, Mondrian dividirá aleatoriamente el área especificada por sus entradas width y height en dos partes, y dibujará un rectangle en cada pieza. Para el nivel 3, Mondrian usa un Mondrian de nivel 2 para subdividir los dos rectángulos más pequeños (para un total de cuatro rectángulos rellenos). El nivel 4 se subdividirá nuevamente, creando ocho rectángulos, y así sucesivamente.
    • La variable fraction contendrá una fracción aleatoria entre 1/5 y 4/5, especificando cómo subdividir el área en este nivel.
    • El algoritmo decide si subdividir el ancho o el alto.
