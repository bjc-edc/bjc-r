Recursive Tree, Part 1

Our goal is to draw a tree like this:

We'll start with a simpler version that shows the technique clearly:

Notice that the tree is like a fractal, a picture that contains smaller versions of itself:

Technically, all the images in this unit are just approximations to fractals because the smaller pictures are simpler than the complete picture. In real fractals, the smaller pictures are just as complete; the smaller and smaller branches go on forever to matter how much you zoom in. In these projects, you can imagine that we have infinitely many smaller and smaller branches.

We'll create a tree block in Snap! It'll start with a move block to draw the trunk of the tree, then a turn block turning left, then another tree block to draw the left smaller tree, and so on.

"Wait!" you may be thinking. "How can we use a tree block inside a tree block? It doesn't exist yet!" That's the big idea for this assignment.

We're going to work up to the complicated tree starting with very simple steps that don't involve tree within tree.
