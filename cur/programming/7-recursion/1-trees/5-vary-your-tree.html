  1. Experiment! Try varying your tree in the following ways:
    You should revert to the original script after each experiment, so that you can always tell exactly what edit in your script relates to the changes on the stage.
    • Change the turn angle.
    • Change the scale factor.
    • Change the number of branches at each level.
    • Change the branching structure.
