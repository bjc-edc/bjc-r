Next we want to change a piece value.



We will only be working with a single entry in the list, the row y. We will be able to use the block below to replace that entry in the list.



This is going to be complicated because we need to do four things for that entry:
  • Copy the letters (if any) that are less than our input x into a new variable
  • Copy the new value onto the end of that variable
  • Copy the letters (if any) that are greater than our input x to the end of that variable
  • Replace that row with the new variable
Before you make the block below "Set" that takes in an x and y value, a list and a new character, try to write the block "copy letters" that you can use in the set function. Copy letters should report the letters from a minimum index to a maximum index and should include the letters at both the minimum and maximum index.
