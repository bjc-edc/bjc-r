The Algorithm

In this activity, you will practice using an algorithm for generating a next word.

In the build-a-story game, you were using your own data (from your brain). A computer only has as much data as you give it. In this activity, we will give the computer a very small amount of data, in a 24-word sentence. This is all the words the computer knows. We'll use this data to practice using an algorithm that generates the next word in a story.

: Algorithm

An algorithm is a set of instructions, or a sequence of steps, usually performed by a computer.

The data:

A curious student enters a science lab.
The science lab smells strange and the science lab feels exciting.
The curious student starts an experiment.
