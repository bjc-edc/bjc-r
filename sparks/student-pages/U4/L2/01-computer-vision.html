In this activity, you'll try categorizing images like a computer does—from just a bunch of pixels.

You learned about pixels in Unit 1 Lab 4 Activity 1: Picture of a Pixel.
Computers don't see the same way people do. Computers just see a bunch of pixels.

  1. Thanks to Zak Kolar, a teacher in Waltham, MA for sharing this project with BJC Sparks!
    Open this Image Classifier Demo. Each image is a recognizable object with its pixels rearranged so you can't tell what it is.
  2. Look at the "Examples" under each class. Sort the "Test Data" in the middle into the “Guesses” sections under each class by dragging and dropping the images.
  3. When you think you've classified the images correctly, click the "Check" button to see the original images unscrambled.
    1. What does a computer "see"?
    2. What are some things we can learn about classifying images from this demo?
