What is L33t?

In the 1980s, the Internet was mostly text-based. Similarly to how today we use emojis ðŸ¦Š ðŸš² and abbreviations (like LMK or LOL) to embellish messages, early Internet users (and some gamers today) used a modified spelling called leet (meaning, "elite") where certain letters are replaced with numbers that look similar (such as replacing "s" with "5" or "e" with "3").

For example, can you read this text?

1 w4nt t0 g0 t0 5ch001 34rly t0d4y.

You're going to build a l33t block that converts just one specific letter to a number (such as just changing "e" to "3").

Your block will need to need to:

  • Split the input text into a list of letters by using .
  • Apply a letter-changing function to every matching letter by using .
  • Join the modified list of letters back into a single text string by using .

You've already learned how to use map to apply a function to every item in a list, and how to use join to join elements of a list into a single text string. The missing ingredient is a letter-changing function.
