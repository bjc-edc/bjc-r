Adding a Stage Background

In this activity, you'll add a background to the stage.

Adding a Stage Background

  1. "Sparks - Interactive Project"
  2. Change the background on the stage.
    Adding a Built-in Stage Background
    1. In the lower right corner of the Snap! window, select the Stage icon (shown right).
    2. Choose "Backgrounds..." from the File () menu.
    3. Choose a background.
    4. Click "Import."
    5. Click "Cancel" to close the dialog box.

  1. You can add more backgrounds if you want. (You'll learn how to import your own backgrounds later.)
  2. If you add more backgrounds, change the background to the first background that you want to appear.
    Changing the Stage Background
    1. If it isn't selected already, Select the Stage icon in the lower right corner of the Snap! window.
      2. The "Backgrounds" tab isn't always visible. It replaces the "Costumes" tab when the stage icon is selected.
    2. Click the "Backgrounds..." tab above the scripting area.
    3. Select a different background, click "Import" to add it to your project, repeat to add more backgrounds, and then click "Cancel" to close.
