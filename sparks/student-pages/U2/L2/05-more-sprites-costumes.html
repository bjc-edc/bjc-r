More with Sprites and Costumes

In this activity, you can add more costumes to your sprites, add more sprites, or even choose the names of your sprites and costumes.

Adding More Costumes

  1. If you want your sprite to have more costumes, add more now. (You'll learn how to import your own costumes on the next page.)
  2. If you added more costumes to your sprite, change the sprite's costume to the first costume that you want it to have.

    Changing a Sprite Costume

    1. Select the Sprite icon if it isn't selected already.
      2. The "Costumes" tab isn't always visible. It replaces the "Backgrounds" tab when a sprite icon is selected.
    2. Click the "Costumes..." tab above the scripting area.
    3. Select a different costume to make it the active costume.

Adding More Sprites

  1. If your project needs more than one sprite, add more, and then give them costume(s) too.

    Adding a Sprite

    • Click on the button just below the stage.
      The new sprite, automatically named "Sprite(2)", will appear on the stage. The pictures of the sprites beneath the stage let you select the sprite you want to work with. (Instructions for renaming your sprites are included below.)

Each Sprite Has Its Own Wardrobe

The costumes you add will appear in the wardrobe of the sprite that you currently have selected. If you add other sprites, you'll have to give them their own costumes. Whichever sprite you have selected, you'll only be able to see the costumes for that sprite.

Each Sprite Also Has Its Own Code

As with costumes, the code you add will appear in the scripting area of the sprite that you currently have selected. If you add other sprites, you'll have to give them their own code, and whichever sprite you have selected, you'll only be able to see the code for that sprite.

Naming Your Sprites

  1. If you want, you can give names to your sprites so you can keep track of them.

    Naming a Sprite

    • Look for the sprite's current name in a gray box above the scripting area. Click inside the name and you'll be able to edit it.

Naming Your Costumes

  1. If you want, you can name or rename your costumes too.

    Renaming a Costumes

    • You can rename any costume by right clicking on the costume in the "Costumes" tab and selecting "rename."
In this activity, you may have added additional costumes and sprites or even named your sprites or costumes.
