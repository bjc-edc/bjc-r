Lab 2: Supervised ML with Teachable Machine  

(from CITE AI Proposal)
a. Introduction to AI and machine learning
b. Computer vision – what do computers “see”?
c. Use Teachable Machine – consider different classes of images you want to build a program around (e.g., awake vs sleeping people, pointing up vs pointing right, etc.) and use Teachable Machine to train a model
d. Import that model into a program and then use the classes of images (and the probabilities associated with new data) to build a project (e.g., it plays an alarm if it sees the user is sleeping, or a character on the screen moves up on the screen if the user points their finger upwards, etc.)
https://experiments.withgoogle.com/teachable-machine

PURPOSE STATEMENT

This project requires use of Google's Teachable Machine and the computer's webcam.
